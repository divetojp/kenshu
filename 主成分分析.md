# 主成分分析（PCA）

高次元データを **情報損失を最小限に抑えながら低次元に圧縮** する手法です。機械学習の前処理・次元削減・可視化で広く使われます。

---

## はじめて読む人へ

50 個の特徴量を持つデータがあるとき、全部をそのまま使うと計算が重くなりますし、2D/3D で可視化もできません。PCA は「一番重要な方向から順に、情報をなるべく保ちながら 2〜3 次元に圧縮する」手術のような手法です。圧縮後のデータでも元のデータの 80〜90% の情報を保持できることが多いです。

### 読む前に押さえること

- [教師なし学習](教師なし学習.md) の「次元削減」の概念
- [線形代数](線形代数.md) の「ベクトルの方向」と「固有値・固有ベクトル」を知っているとより深く理解できます（必須ではない）
- [特徴量エンジニアリング](特徴量エンジニアリング.md) の StandardScaler（PCA 前にスケーリングが必須）

### 読み終えたら説明できること

- 「分散が大きい方向 = 情報量が多い」という PCA の直感を説明できる
- 寄与率と累積寄与率の意味を説明できる
- PCA と t-SNE・因子分析の使い分けを説明できる

---

## 仕組み

### なぜ「分散が最大になる方向」なのか

まず「なぜ次元を削減したいのか」から考えます。

!!! info ""
    問題：5 つの試験（数学・英語・理科・社会・国語）の点数で生徒を分析したい
          5 次元のデータは可視化も理解も難しい
    
    目標：情報の損失を最小限に抑えながら、2〜3 次元に圧縮したい

重要なのは「**情報量 = データのばらつき（分散）**」という考え方です。

!!! info ""
    例えば、全員が同じ数学の点数なら…
      → 数学の点数から何も学べない（情報量ゼロ）
      → この列は削除しても OK
    
    逆に、数学の点数がばらついているなら…
      → 生徒ごとの違いが見える（情報量が多い）
      → この列は重要

PCA は「全変数の中で **最もデータがばらつく方向（軸）** を見つけ、その軸を新しい座標系として使う」手法です。分散が大きい方向へ射影すれば、元のデータのばらつき（違い）を最大限保持できます。

!!! info ""
    元のデータ（2 次元の例）:
      ↑ y（英語）
      │  ・
      │・   ・
      │  ・
      │・  ・
      └────────→ x（数学）
    
    数学と英語は相関している（数学が得意な人は英語も得意な傾向）
    → データは「右上から左下方向」に広く分布している
    
    PCA が見つける第1主成分:
                    ↗ （データが最も広がる方向 = 第1主成分）
                  ↗
                ↗
    
    この方向に射影すれば、2 次元の情報を 1 次元で 90% 表せる

数学的には、共分散行列 $A$ の **固有ベクトル**（最大固有値に対応）が第1主成分の方向になります。固有値方程式 $Av = \lambda v$ の解として求まります（[線形代数](線形代数.md) の固有値分解と対応）。

```python
import numpy as np
import pandas as pd
import matplotlib.pyplot as plt
from sklearn.preprocessing import StandardScaler
from sklearn.decomposition import PCA
from sklearn.datasets import load_iris

# Iris データセット（4 変数 → 2 次元に削減）
data = load_iris()
X = data.data
y = data.target
feature_names = data.feature_names

# 標準化（PCA 前の必須ステップ）
scaler = StandardScaler()
X_scaled = scaler.fit_transform(X)

# PCA
pca = PCA()
pca.fit(X_scaled)

# 各主成分の寄与率
explained = pca.explained_variance_ratio_
cumulative = np.cumsum(explained)
print("寄与率:", [f"{e:.3f}" for e in explained])
print("累積寄与率:", [f"{c:.3f}" for c in cumulative])
# [0.728, 0.229, 0.037, 0.006]  ← 2 主成分で 95.7% を説明
```

---

## 次元数の選択（スクリープロット）

```python
fig, axes = plt.subplots(1, 2, figsize=(12, 4))

# スクリープロット（各主成分の固有値）
axes[0].bar(range(1, len(explained)+1), pca.explained_variance_)
axes[0].set_xlabel("主成分"); axes[0].set_ylabel("固有値（分散）")
axes[0].set_title("スクリープロット")

# 累積寄与率
axes[1].plot(range(1, len(cumulative)+1), cumulative, 'bo-')
axes[1].axhline(0.95, color='red', linestyle='--', label='95%')
axes[1].set_xlabel("主成分数"); axes[1].set_ylabel("累積寄与率")
axes[1].set_title("累積寄与率")
axes[1].legend()

plt.tight_layout()
plt.show()
```

**次元数の選び方：**
- 累積寄与率が 80〜95% になる主成分数を選ぶ
- スクリープロットで固有値が急激に落ちる「肘」の点を選ぶ

---

## 2 次元への圧縮と可視化

```python
pca_2d = PCA(n_components=2)
X_pca = pca_2d.fit_transform(X_scaled)

plt.figure(figsize=(8, 5))
scatter = plt.scatter(X_pca[:, 0], X_pca[:, 1], c=y, cmap='viridis', alpha=0.7)
plt.colorbar(scatter, label='品種')
plt.xlabel(f"PC1 ({pca_2d.explained_variance_ratio_[0]:.1%})")
plt.ylabel(f"PC2 ({pca_2d.explained_variance_ratio_[1]:.1%})")
plt.title("Iris: PCA による 2 次元可視化")
plt.show()
```

## ローディング行列（主成分と元変数の関係）

```python
loadings = pd.DataFrame(
    pca_2d.components_.T,
    index=feature_names,
    columns=["PC1", "PC2"]
)
print(loadings)
```

| ローディングの絶対値 | 解釈 |
|------------------|------|
| 0.7 以上 | その主成分に強く寄与する変数 |
| 0.3 〜 0.7 | 中程度 |
| 0.3 未満 | ほとんど寄与しない |
