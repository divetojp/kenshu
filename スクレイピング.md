# スクレイピング（Web データ収集）

Web サイトから自動的にデータを取得する技術です。「公開されているデータをプログラムで収集する」ことで、分析用データセットを自分で作れるようになります。ただし**法的・倫理的な制約**があり、必ず事前確認が必要です。

---

## 法的・倫理的注意点（必読）

### robots.txt を確認する

`https://example.com/robots.txt` にアクセスして、スクレイピングが許可されているか確認します。

!!! info ""
    ```text
    # robots.txt の例
    User-agent: *         # すべてのボットに対して
    Disallow: /admin/     # /admin/ 以下は禁止
    Disallow: /private/   # /private/ 以下は禁止
    Allow: /public/       # /public/ は許可

    Crawl-delay: 10       # 10秒以上間隔を空けること
    ```
```python
import urllib.robotparser

def can_scrape(url: str, user_agent: str = "*") -> bool:
    """robots.txt でスクレイピング可能か確認する"""
    from urllib.parse import urlparse
    base = f"{urlparse(url).scheme}://{urlparse(url).netloc}"
    rp = urllib.robotparser.RobotFileParser()
    rp.set_url(f"{base}/robots.txt")
    rp.read()
    return rp.can_fetch(user_agent, url)

print(can_scrape("https://www.python.org/"))  # True/False
```

### 利用規約を確認する

robots.txt に加えて、サービスの**利用規約（Terms of Service）**も確認します。

!!! info ""
    **✗ 禁止の例（利用規約に明記されている場合)**

    「自動化された手段によるデータ収集を禁止します」
    「スクレイピング・クロールを禁止します」

    **✓ 許可されている例**

    「非営利・学術目的での利用は許可します」
    「API が提供されている場合、API を優先してください」
### 守るべきルール

!!! info ""
    □ robots.txt で禁止されているパスにはアクセスしない
    □ 利用規約を読んでスクレイピング禁止でないか確認する
    □ アクセス間隔を空ける（最低 1〜2 秒、Crawl-delay に従う）
    □ User-Agent に自分の連絡先を含める
    □ 個人情報（氏名・住所・電話番号）を収集しない
    □ 収集したデータを許可なく再配布しない
    □ API がある場合は API を使う（スクレイピングより公式）

## 基本：requests でページを取得

```python
import requests
import time

headers = {
    "User-Agent": "MyResearchBot/1.0 (contact@example.com)"
    # 自分の連絡先を含めることでサーバー管理者が連絡できる
}

url = "https://books.toscrape.com/"   # スクレイピング練習用サイト

response = requests.get(url, headers=headers, timeout=10)

# ステータスコードの確認
print(response.status_code)   # 200 = 成功
response.raise_for_status()   # 4xx・5xx でも例外を発生させる

# エンコーディング
response.encoding = response.apparent_encoding  # 文字化け対策
html = response.text

print(html[:500])
```

## HTML の解析：BeautifulSoup

```python
from bs4 import BeautifulSoup
import requests

response = requests.get("https://books.toscrape.com/", timeout=10)
soup = BeautifulSoup(response.text, "html.parser")

# タグで検索
title = soup.find("title")
print(title.text)  # "All products | Books to Scrape - Sandbox"

# CSS セレクタで検索（最も汎用的）
# article.product_pod h3 a → article タグ（クラス product_pod）内の h3 内の a タグ
books = soup.select("article.product_pod h3 a")
for book in books[:5]:
    print(book.get("title"))   # 属性の取得

# クラス名で検索
prices = soup.find_all("p", class_="price_color")
for price in prices[:5]:
    print(price.text)  # "£51.77"
```

### CSS セレクタのチートシート

| セレクタ | 意味 | 例 |
|--------|------|-----|
| `div` | タグ名 | `soup.select("div")` |
| `.classname` | クラス | `soup.select(".price")` |
| `#idname` | ID | `soup.select("#main")` |
| `div.class` | タグ + クラス | `soup.select("p.title")` |
| `div > p` | 直接の子要素 | `soup.select("ul > li")` |
| `div p` | 子孫要素（ネスト） | `soup.select("div p")` |
| `a[href]` | 属性の存在 | `soup.select("a[href]")` |
| `a[href^="https"]` | 属性の値が前方一致 | `soup.select('a[href^="https"]')` |

## 実践：複数ページをスクレイピング

```python
import requests
from bs4 import BeautifulSoup
import time
import csv

BASE_URL = "https://books.toscrape.com/catalogue/"

def scrape_page(url: str) -> list[dict]:
    """1ページ分の本の情報を取得"""
    response = requests.get(url, timeout=10)
    response.raise_for_status()
    soup = BeautifulSoup(response.text, "html.parser")

    books = []
    for article in soup.select("article.product_pod"):
        title = article.select_one("h3 a")["title"]
        price = article.select_one(".price_color").text.strip()
        rating_class = article.select_one(".star-rating")["class"][1]
        rating_map = {"One": 1, "Two": 2, "Three": 3, "Four": 4, "Five": 5}
        rating = rating_map.get(rating_class, 0)

        books.append({"title": title, "price": price, "rating": rating})
    return books

def get_next_page(soup: BeautifulSoup, current_url: str) -> str | None:
    """次のページの URL を取得（なければ None）"""
    next_btn = soup.select_one("li.next a")
    if next_btn:
        return BASE_URL + next_btn["href"]
    return None

# 最大 3 ページ収集
all_books = []
url = "https://books.toscrape.com/catalogue/page-1.html"
max_pages = 3

for page_num in range(1, max_pages + 1):
    print(f"ページ {page_num} を取得中...")
    response = requests.get(url, timeout=10)
    soup = BeautifulSoup(response.text, "html.parser")

    all_books.extend(scrape_page(url))

    url = get_next_page(soup, url)
    if url is None:
        break
    time.sleep(1.5)   # サーバーに負荷をかけないよう待機

print(f"合計 {len(all_books)} 件取得")

# CSV に保存
with open("books.csv", "w", newline="", encoding="utf-8") as f:
    writer = csv.DictWriter(f, fieldnames=["title", "price", "rating"])
    writer.writeheader()
    writer.writerows(all_books)
```

## 動的サイトには Playwright / Selenium

JavaScript で描画されるサイト（React・Vue などの SPA）は requests では取得できません。

```python
# pip install playwright
# playwright install chromium

from playwright.sync_api import sync_playwright

with sync_playwright() as p:
    browser = p.chromium.launch(headless=True)
    page = browser.new_page()
    page.goto("https://example.com")

    # JavaScript の実行を待つ
    page.wait_for_selector(".dynamic-content")

    # テキストを取得
    text = page.locator(".dynamic-content").text_content()
    print(text)

    browser.close()
```

**注意：** 動的サイトのスクレイピングはサーバーへの負荷が大きく、robots.txt で禁止されている場合が多いです。API が提供されているなら必ず API を使いましょう。

## スクレイピング vs API

| 観点 | スクレイピング | API |
|------|------------|-----|
| 安定性 | サイトの HTML 変更で壊れる | 仕様変更まで安定 |
| 速度 | 遅い（HTML 全体を取得） | 速い（必要なデータのみ） |
| 許可 | 利用規約で禁止の場合あり | 公式に提供 |
| 取得できるデータ | サイト上に表示されているもの | 設計された範囲のみ |

**API があれば必ず API を使いましょう。**
