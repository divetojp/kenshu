# 最適化理論

「制約のある条件下で最良の解を見つける」数学的フレームワークです。[微分・最適化基礎](微分・最適化基礎.md) が「勾配降下法でパラメータを更新する」実装寄りの内容なら、このページは「**なぜその解が最適か**」を証明する理論を扱います。SVM の導出・制約付き機械学習・凸最適化の基礎になります。

## 最適化問題の一般形

$$
\min_{x} \quad f(x) \quad \text{subject to} \quad
\begin{cases}
g_i(x) \leq 0, & i = 1, \ldots, m \\
h_j(x) = 0, & j = 1, \ldots, p
\end{cases}
$$

- $f(x)$：目的関数（最小化したい値）
- $g_i(x) \leq 0$：不等式制約
- $h_j(x) = 0$：等式制約

## 凸関数と凸最適化

### 凸関数の定義

関数 $f$ が凸関数 $\Leftrightarrow$ 任意の $x, y$ と $\lambda \in [0,1]$ に対して：

$$
f(\lambda x + (1-\lambda)y) \leq \lambda f(x) + (1-\lambda)f(y)
$$

**直感：「弦が関数の上にある」**

```python
import numpy as np
import matplotlib.pyplot as plt

x = np.linspace(-2, 2, 300)

fig, axes = plt.subplots(1, 2, figsize=(10, 4))

# 凸関数の例: f(x) = x²
axes[0].plot(x, x**2, label="f(x) = x²（凸関数）")
x1, x2 = -1.5, 1.0
y1, y2 = x1**2, x2**2
axes[0].plot([x1, x2], [y1, y2], 'r--', label="弦（必ず関数の上）")
axes[0].legend(); axes[0].set_title("凸関数")

# 非凸関数の例
axes[1].plot(x, x**4 - 2*x**2, label="f(x) = x⁴ - 2x²（非凸）")
x1, x2 = -1.5, 1.5
y1 = x1**4 - 2*x1**2
y2 = x2**4 - 2*x2**2
axes[1].plot([x1, x2], [y1, y2], 'r--', label="弦（関数の下に来ることも）")
axes[1].legend(); axes[1].set_title("非凸関数（局所最適が複数）")

plt.tight_layout()
plt.savefig("convex.png", dpi=150)
```

### 凸最適化の強力な性質

**凸問題では局所最適解 = 大域最適解**

!!! info ""
    ```
    非凸問題: 複数の谷があり、局所最適に罠にはまる可能性
      f(x)
       |
       |    ↑局所最適（谷 A）  ↑大域最適（谷 B）
       |   / \___/   \___/
       |  /                \
       ↑─────────────────────→ x
    
    凸問題: 谷は 1 つしかない
      f(x)
       |
       |      ↑大域最適（唯一の解）
       |     /V\
       |    /   \
       ↑──────────────────────→ x
    ```

```python
# よく使う凸関数の例
convex_examples = {
    "f(x) = x²": lambda x: x**2,
    "f(x) = |x|": lambda x: np.abs(x),
    "f(x) = e^x": lambda x: np.exp(x),
    "f(x) = -log(x)（x>0）": lambda x: -np.log(x),

# Lasso 正則化: L(w) = MSE + α||w||₁ → 全体として凸（スパース解）
```

## ラグランジュ乗数法

**等式制約付き** 最適化を解く手法です。

問題：$\min f(x)$ subject to $h(x) = 0$

**ラグランジュ関数：**

$$
\mathcal{L}(x, \lambda) = f(x) + \lambda h(x)
$$

**最適性条件：**

$$
\nabla_x \mathcal{L} = \nabla f(x^*) + \lambda^* \nabla h(x^*) = 0
$$

### 例：球面上の最大値問題

$\max x + y$ subject to $x^2 + y^2 = 1$

```python
import sympy as sp

x, y, lam = sp.symbols('x y lambda')

f = x + y                       # 目的関数（最大化）
h = x**2 + y**2 - 1            # 制約 h(x,y) = 0

L = f + lam * h                  # ラグランジュ関数

# 停留点の条件（各変数で偏微分 = 0）
eqs = [
    sp.diff(L, x),   # ∂L/∂x = 0
    sp.diff(L, y),   # ∂L/∂y = 0
    sp.diff(L, lam), # ∂L/∂λ = 0 ← 制約条件と同じ
]

solution = sp.solve(eqs, [x, y, lam])
print(solution)
# [(1/√2, 1/√2, -1/√2), (-1/√2, -1/√2, 1/√2)]
# → 最大値は x=y=1/√2 のとき f = √2
```

## KKT 条件（不等式制約）

**不等式制約付き** 最適化の最適性条件（Karush-Kuhn-Tucker）：

問題：$\min f(x)$ subject to $g_i(x) \leq 0$, $h_j(x) = 0$

**KKT 条件（必要条件）：**

$$
\nabla f(x^*) + \sum_i \mu_i \nabla g_i(x^*) + \sum_j \lambda_j \nabla h_j(x^*) = 0
\quad \text{（停留性）}
$$

$$
\mu_i \geq 0 \quad \text{（双対可能性）}
$$

$$
\mu_i g_i(x^*) = 0 \quad \text{（相補性）}
$$

**相補性条件の意味：** $\mu_i > 0$ なら $g_i(x^*) = 0$（制約が能動）、$g_i(x^*) < 0$ なら $\mu_i = 0$（制約が非能動）。

## SVM との接続

SVM の最適化問題：

$$
\min_{w, b} \frac{1}{2}\|w\|^2 \quad \text{subject to} \quad y_i(w^\top x_i + b) \geq 1
$$

KKT 条件を適用すると：

$$
w = \sum_i \alpha_i y_i x_i
$$

$\alpha_i > 0$ となるのは**制約が能動**なデータ点のみ——これが**サポートベクター**です。マージン境界上にある点だけが $w$ の決定に寄与します。

```python
from sklearn.svm import SVC
from sklearn.datasets import make_classification
import numpy as np

X, y = make_classification(n_samples=50, n_features=2, n_redundant=0,
                            n_clusters_per_class=1, random_state=42)
y = 2*y - 1  # {0,1} → {-1, 1}

svm = SVC(kernel='linear', C=1e6)  # ハードマージン近似
svm.fit(X, y)

print(f"サポートベクターの数: {len(svm.support_vectors_)}")
print(f"全データ {len(X)} 件中 {len(svm.support_vectors_)} 件だけが w を決定する")
# → KKT 相補性条件の現れ
```

## 双対問題

元の最小化問題（主問題）に対して、下界を与える最大化問題（双対問題）が対応します。

$$
\text{主問題}: \min_x f(x) \quad \Rightarrow \quad \text{双対問題}: \max_\lambda g(\lambda) \leq \min_x f(x)
$$

**強双対性（凸問題では成立）：** 主問題と双対問題の最適値が一致します。

SVM では双対問題を解く方が効率的なことがあります：

$$
\max_\alpha \sum_i \alpha_i - \frac{1}{2}\sum_{i,j} \alpha_i \alpha_j y_i y_j x_i^\top x_j
$$

この形では $x_i^\top x_j$（内積）しか登場しないため、**カーネルトリック**が使えます。

## 数学的導出

### ラグランジュ乗数法の直感的な証明

等式制約 $h(x) = 0$ のもとで $f(x)$ を最小化するとします。

最適点 $x^*$ において、$f$ の勾配 $\nabla f(x^*)$ は制約面 $h(x) = 0$ の接線空間に対して **垂直でなければならない** ことを示します。

もし $\nabla f(x^*)$ が接線空間に成分を持つなら、制約面に沿って $f$ を減らせる方向に動けることになり、$x^*$ が最適点であることに矛盾します。

制約面の法線方向は $\nabla h(x^*)$ ですから：

$$
\nabla f(x^*) = -\lambda^* \nabla h(x^*) \quad \Leftrightarrow \quad \nabla f(x^*) + \lambda^* \nabla h(x^*) = 0
$$

この条件を一つの関数 $\mathcal{L}(x, \lambda) = f(x) + \lambda h(x)$ の停留条件として書いたものがラグランジュ乗数法です。

---

### SVM 主問題から双対問題への完全導出

**主問題：**

$$
\min_{w,b} \frac{1}{2}\|w\|^2 \quad \text{s.t.} \quad y_i(w^\top x_i + b) \geq 1, \; i=1,\ldots,n
$$

制約 $1 - y_i(w^\top x_i + b) \leq 0$ に対してラグランジュ乗数 $\alpha_i \geq 0$ を導入します：

$$
\mathcal{L}(w, b, \alpha) = \frac{1}{2}\|w\|^2 - \sum_i \alpha_i [y_i(w^\top x_i + b) - 1]
$$

**KKT 停留条件：**

$$
\frac{\partial \mathcal{L}}{\partial w} = w - \sum_i \alpha_i y_i x_i = 0 \quad \Rightarrow \quad w^* = \sum_i \alpha_i y_i x_i \quad \cdots (1)
$$

$$
\frac{\partial \mathcal{L}}{\partial b} = -\sum_i \alpha_i y_i = 0 \quad \Rightarrow \quad \sum_i \alpha_i y_i = 0 \quad \cdots (2)
$$

**(1)(2) をラグランジュ関数に代入して双対関数を導出：**

$$
\mathcal{L} = \frac{1}{2}\|w^*\|^2 - \sum_i \alpha_i y_i (w^*)^\top x_i - b \underbrace{\sum_i \alpha_i y_i}_{=0} + \sum_i \alpha_i
$$

$$
= \frac{1}{2}\sum_{i,j}\alpha_i\alpha_j y_i y_j x_i^\top x_j - \sum_{i,j}\alpha_i\alpha_j y_i y_j x_i^\top x_j + \sum_i \alpha_i
$$

$$
\boxed{g(\alpha) = \sum_i \alpha_i - \frac{1}{2}\sum_{i,j}\alpha_i\alpha_j y_i y_j x_i^\top x_j}
$$

**双対問題：**

$$
\max_\alpha \; g(\alpha) \quad \text{s.t.} \quad \alpha_i \geq 0, \quad \sum_i \alpha_i y_i = 0
$$

この形では $x_i^\top x_j$（内積）しか現れないため、$k(x_i, x_j) = \phi(x_i)^\top \phi(x_j)$ に置き換えるだけでカーネルトリックが適用できます。

---

### 凸関数の一次条件（勾配不等式）

微分可能な凸関数 $f$ は次を満たします：

$$
f(y) \geq f(x) + \nabla f(x)^\top (y - x) \quad \forall x, y
$$

**証明：** 凸性の定義から $f(\lambda y + (1-\lambda)x) \leq \lambda f(y) + (1-\lambda)f(x)$ 。整理すると：

$$
\frac{f(x + \lambda(y-x)) - f(x)}{\lambda} \leq f(y) - f(x)
$$

$\lambda \to 0$ の極限をとると左辺 → $\nabla f(x)^\top (y-x)$ となり、証明完了。

**重要な帰結：** 凸関数では $\nabla f(x^*) = 0$ が大域最小値の必要十分条件です。
