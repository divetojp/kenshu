# 異常検知

**「普通ではないデータ」を自動で見つける**手法です。製造業の不良品検出・金融の不正取引検知・インフラ障害の予兆検知・医療の異常診断——正常データは大量にあるが異常データは少ない・あるいは未知という設定に強みがあります。

## 異常の種類

!!! info ""
    ```
    点異常（Point Anomaly）:
      1 件のデータが全体から外れている
      例: 「平均気温 25℃のデータ群の中に 80℃の値がある」
    
    文脈異常（Contextual Anomaly）:
      文脈（時刻・場所）の中で異常だが全体では正常範囲
      例: 「7月に気温 25℃は正常、12月に 25℃は異常」
    
    集合異常（Collective Anomaly）:
      個々は正常だが、集まると異常
      例: 「ECG の各点は正常値だが、そのパターンが不整脈を示す」
    ```

## 統計的アプローチ（シンプルな方法）

### Z スコア・IQR

```python
import numpy as np
import pandas as pd

rng = np.random.default_rng(42)
data = np.concatenate([
    rng.normal(0, 1, 990),   # 正常データ
    rng.normal(5, 0.5, 10),  # 異常データ（平均 5 の外れ値）
])

# Z スコア法（標準偏差の何倍外れているか）
z_scores = np.abs((data - data.mean()) / data.std())
anomalies_z = np.where(z_scores > 3)[0]
print(f"Z スコア法で検出した異常件数: {len(anomalies_z)}")

# IQR 法（四分位範囲を使う）
Q1, Q3 = np.percentile(data, [25, 75])
IQR = Q3 - Q1
lower, upper = Q1 - 1.5 * IQR, Q3 + 1.5 * IQR
anomalies_iqr = np.where((data < lower) | (data > upper))[0]
print(f"IQR 法で検出した異常件数: {len(anomalies_iqr)}")
```

## Isolation Forest

**「異常なデータほど少ない分割で孤立する」**という直感に基づく手法です。ランダムな分割を繰り返し、早く孤立するほど異常スコアが高くなります。

```python
from sklearn.ensemble import IsolationForest
from sklearn.datasets import make_blobs
import numpy as np
import matplotlib.pyplot as plt

# 正常データ + 外れ値を生成
rng = np.random.default_rng(42)
X_normal, _ = make_blobs(n_samples=300, centers=1, cluster_std=0.5, random_state=42)
X_outliers = rng.uniform(-4, 4, (20, 2))
X = np.vstack([X_normal, X_outliers])

# Isolation Forest
clf = IsolationForest(contamination=0.06, random_state=42)  # contamination: 異常の割合の推定
labels = clf.fit_predict(X)   # 1: 正常、-1: 異常
scores = clf.score_samples(X) # 低いほど異常（負の値）

print(f"検出した異常件数: {(labels == -1).sum()}")

# 可視化
colors = ['red' if l == -1 else 'steelblue' for l in labels]
plt.scatter(X[:, 0], X[:, 1], c=colors, alpha=0.7)
plt.title("Isolation Forest による異常検知（赤 = 異常）")
plt.show()
```

**特徴：** 次元数が多くても高速・精度が高い。`contamination` パラメータで期待する異常率を調整します。

## LOF（Local Outlier Factor）

**局所密度**を使って異常を検出します。周囲のデータと比べて密度が著しく低い点を異常とみなします。

```python
from sklearn.neighbors import LocalOutlierFactor

lof = LocalOutlierFactor(n_neighbors=20, contamination=0.06)
labels_lof = lof.fit_predict(X)
lof_scores = -lof.negative_outlier_factor_   # 高いほど異常

print(f"LOF で検出した異常件数: {(labels_lof == -1).sum()}")
```

**IsolationForest vs LOF の使い分け：**

| 観点 | IsolationForest | LOF |
|------|----------------|-----|
| 速度 | 速い | データ量に応じて遅くなる |
| 大域 vs 局所 | 大域的な異常に強い | 局所的な密度異常に強い |
| パラメータ | `contamination` | `n_neighbors` |

## Autoencoder による異常検知

**正常データだけで Autoencoder を学習**し、再構成誤差が大きいデータを異常とみなします。

```python
import torch
import torch.nn as nn
import numpy as np

class Autoencoder(nn.Module):
    def __init__(self, input_dim: int, latent_dim: int = 8):
        super().__init__()
        self.encoder = nn.Sequential(
            nn.Linear(input_dim, 32),
            nn.ReLU(),
            nn.Linear(32, latent_dim),
        )
        self.decoder = nn.Sequential(
            nn.Linear(latent_dim, 32),
            nn.ReLU(),
            nn.Linear(32, input_dim),
        )

    def forward(self, x):
        return self.decoder(self.encoder(x))

# 正常データのみで学習
rng = np.random.default_rng(42)
X_normal = rng.normal(0, 1, (1000, 20)).astype(np.float32)
X_tensor = torch.FloatTensor(X_normal)

model = Autoencoder(input_dim=20, latent_dim=4)
optimizer = torch.optim.Adam(model.parameters(), lr=1e-3)

for epoch in range(100):
    recon = model(X_tensor)
    loss = nn.MSELoss()(recon, X_tensor)
    optimizer.zero_grad(); loss.backward(); optimizer.step()

print(f"最終損失: {loss.item():.4f}")

# 推論時: 再構成誤差でスコアリング
def anomaly_score(model, x):
    model.eval()
    with torch.no_grad():
        recon = model(torch.FloatTensor(x))
        return ((torch.FloatTensor(x) - recon) ** 2).mean(dim=1).numpy()

# 正常データと異常データの再構成誤差を比較
X_test_normal  = rng.normal(0, 1, (100, 20)).astype(np.float32)
X_test_anomaly = rng.normal(5, 1, (20, 20)).astype(np.float32)   # 分布が違う異常

scores_normal  = anomaly_score(model, X_test_normal)
scores_anomaly = anomaly_score(model, X_test_anomaly)

print(f"正常データの平均再構成誤差: {scores_normal.mean():.4f}")
print(f"異常データの平均再構成誤差: {scores_anomaly.mean():.4f}")
# 異常のほうが大きくなる
```

## 時系列の異常検知

### 統計的な方法（移動平均 + 標準偏差）

```python
import pandas as pd
import numpy as np
import matplotlib.pyplot as plt

rng = np.random.default_rng(42)
n = 200
t = np.arange(n)

# 正常: トレンド + 季節性 + ノイズ
signal = 0.1 * t + 5 * np.sin(2 * np.pi * t / 30) + rng.normal(0, 1, n)
# 異常を注入
signal[80] += 15
signal[140:145] += 10

df = pd.Series(signal)

# ローリング統計による検出
window = 20
rolling_mean = df.rolling(window).mean()
rolling_std  = df.rolling(window).std()
threshold    = 3   # 3σ ルール

upper = rolling_mean + threshold * rolling_std
lower = rolling_mean - threshold * rolling_std
anomalies = df[(df > upper) | (df < lower)]

plt.figure(figsize=(12, 4))
plt.plot(df, label='データ', alpha=0.7)
plt.plot(rolling_mean, label='移動平均', color='orange')
plt.fill_between(range(n), upper, lower, alpha=0.2, color='orange', label='3σ 帯域')
plt.scatter(anomalies.index, anomalies, color='red', zorder=5, label='異常')
plt.legend(); plt.title('時系列の異常検知（3σ法）')
plt.show()
```

## 評価指標

異常検知は **正解ラベルがないことが多い**ため評価が難しいです。ラベルがある場合：

```python
from sklearn.metrics import classification_report, roc_auc_score
import numpy as np

# ラベルがある場合の評価
y_true = np.array([0]*980 + [1]*20)  # 0: 正常、1: 異常
y_pred = (labels_lof == -1).astype(int)

print(classification_report(y_true, y_pred, target_names=["正常", "異常"]))
# 異常が少ないのでクラス不均衡に注意
# → precision・recall・F1 で評価（accuracyだけでは不十分）

# AUC-ROC（スコアがある場合）
auc = roc_auc_score(y_true, lof_scores)
print(f"AUC-ROC: {auc:.3f}")
```
