# 多変量解析

**複数の変数を同時に扱い、変数間の構造・パターンを明らかにする統計手法の総称**です。データの次元削減・グルーピング・分類・関係構造の把握に使います。

---

## はじめて読む人へ

現実のデータは多くの変数を持ちます。多変量解析は「変数が多すぎて直接見づらいデータ」の構造を見えやすくする道具です。ML の文脈で「次元削減」「クラスタリング」と呼ばれる技術の統計的背景がここにあります。

### 読む前に押さえること

- PCA（主成分分析）は、情報を損なわず次元を削減します。元の変数の「線形結合」で新しい軸（主成分）を作ります。
- 因子分析は PCA に似ていますが、「観測できない潜在因子がデータを生成した」という仮定のもとで解釈します。
- 判別分析はグループ間の差を最大化する方向を見つけ、新しいデータを分類します。

### 読み終えたら説明できること

- PCA でどのくらい次元を削減すればよいか、寄与率から判断できる。
- 因子分析と PCA の目的の違いを説明できる。
- LDA と PCA の違いを説明できる。

---

## 主成分分析（PCA）

高次元データを **情報損失を最小限に抑えながら低次元に圧縮** する手法です。

### 仕組み

#### なぜ「分散が最大になる方向」なのか

まず「なぜ次元を削減したいのか」から考えます。

```
問題：5 つの試験（数学・英語・理科・社会・国語）の点数で生徒を分析したい
      5 次元のデータは可視化も理解も難しい

目標：情報の損失を最小限に抑えながら、2〜3 次元に圧縮したい
```

重要なのは「**情報量 = データのばらつき（分散）**」という考え方です。

```
例えば、全員が同じ数学の点数なら…
  → 数学の点数から何も学べない（情報量ゼロ）
  → この列は削除しても OK

逆に、数学の点数がばらついているなら…
  → 生徒ごとの違いが見える（情報量が多い）
  → この列は重要
```

PCA は「全変数の中で **最もデータがばらつく方向（軸）** を見つけ、その軸を新しい座標系として使う」手法です。「なぜ分散が最大になる方向を選ぶのか？」という問いに答えておきましょう。分散が小さい方向にデータを射影すると、データ同士の違いがつぶれてしまいます。逆に分散が大きい方向へ射影すれば、元のデータのばらつき（違い）を最大限保持できます。情報量 ≈ ばらつき、という考え方のもとで「最も情報を残せる軸」を選んでいます。

```
元のデータ（2 次元の例）:
  ↑ y（英語）
  │  ・
  │・   ・
  │  ・
  │・  ・
  └────────→ x（数学）

数学と英語は相関している（数学が得意な人は英語も得意な傾向）
→ データは「右上から左下方向」に広く分布している

PCA が見つける第1主成分:
                ↗ （データが最も広がる方向 = 第1主成分）
              ↗
            ↗

この方向に射影すれば、2 次元の情報を 1 次元で 90% 表せる
```

数学的には、共分散行列 $A$ の **固有ベクトル**（最大固有値に対応）が第1主成分の方向になります。固有値方程式 $Av = \lambda v$ の解として求まります（[線形代数](線形代数)の固有値分解と対応）。

```python
import numpy as np
import pandas as pd
import matplotlib.pyplot as plt
from sklearn.preprocessing import StandardScaler
from sklearn.decomposition import PCA
from sklearn.datasets import load_iris

# Iris データセット（4 変数 → 2 次元に削減）
data = load_iris()
X = data.data
y = data.target
feature_names = data.feature_names

# 標準化（PCA 前の必須ステップ）
scaler = StandardScaler()
X_scaled = scaler.fit_transform(X)

# PCA
pca = PCA()
pca.fit(X_scaled)

# 各主成分の寄与率
explained = pca.explained_variance_ratio_
cumulative = np.cumsum(explained)
print("寄与率:", [f"{e:.3f}" for e in explained])
print("累積寄与率:", [f"{c:.3f}" for c in cumulative])
# [0.728, 0.229, 0.037, 0.006]  ← 2 主成分で 95.7% を説明
```

### 次元数の選択（スクリープロット）

```python
fig, axes = plt.subplots(1, 2, figsize=(12, 4))

# スクリープロット（各主成分の固有値）
axes[0].bar(range(1, len(explained)+1), pca.explained_variance_)
axes[0].set_xlabel("主成分"); axes[0].set_ylabel("固有値（分散）")
axes[0].set_title("スクリープロット")

# 累積寄与率
axes[1].plot(range(1, len(cumulative)+1), cumulative, 'bo-')
axes[1].axhline(0.95, color='red', linestyle='--', label='95%')
axes[1].set_xlabel("主成分数"); axes[1].set_ylabel("累積寄与率")
axes[1].set_title("累積寄与率")
axes[1].legend()

plt.tight_layout()
plt.show()
```

**次元数の選び方：**
- 累積寄与率が 80〜95% になる主成分数を選ぶ
- スクリープロットで固有値が急激に落ちる「肘」の点を選ぶ

### 2 次元への圧縮と可視化

```python
pca_2d = PCA(n_components=2)
X_pca = pca_2d.fit_transform(X_scaled)

plt.figure(figsize=(8, 5))
scatter = plt.scatter(X_pca[:, 0], X_pca[:, 1], c=y, cmap='viridis', alpha=0.7)
plt.colorbar(scatter, label='品種')
plt.xlabel(f"PC1 ({pca_2d.explained_variance_ratio_[0]:.1%})")
plt.ylabel(f"PC2 ({pca_2d.explained_variance_ratio_[1]:.1%})")
plt.title("Iris: PCA による 2 次元可視化")
plt.show()
```

### ローディング行列（主成分と元変数の関係）

```python
loadings = pd.DataFrame(
    pca_2d.components_.T,
    index=feature_names,
    columns=["PC1", "PC2"]
)
print(loadings)
# 各変数が各主成分にどれだけ寄与するか（-1 〜 1）
```

| ローディングの絶対値 | 解釈 |
|------------------|------|
| 0.7 以上 | その主成分に強く寄与する変数 |
| 0.3 〜 0.7 | 中程度 |
| 0.3 未満 | ほとんど寄与しない |

## 因子分析（Factor Analysis）

PCA が「次元削減の道具」なのに対し、因子分析は **潜在的な共通因子** を発見するための統計モデルです。

$$
x_i = \lambda_{i1} F_1 + \lambda_{i2} F_2 + \cdots + \varepsilon_i
$$

- $F_j$：潜在因子（直接観測できない）
- $\lambda_{ij}$：因子負荷量（変数 $i$ と因子 $j$ の関係の強さ）
- $\varepsilon_i$：独自因子（その変数固有の部分）

```python
from sklearn.decomposition import FactorAnalysis
from sklearn.datasets import load_iris

data = load_iris()
X_scaled = StandardScaler().fit_transform(data.data)

fa = FactorAnalysis(n_components=2, random_state=42)
fa.fit(X_scaled)

# 因子負荷量行列
loadings_fa = pd.DataFrame(
    fa.components_.T,
    index=data.feature_names,
    columns=["Factor1", "Factor2"]
)
print(loadings_fa.round(3))
```

### PCA vs 因子分析

| 観点 | PCA | 因子分析 |
|------|-----|---------|
| 目的 | 次元削減・圧縮 | 潜在構造の発見・解釈 |
| モデル | 統計モデルなし（行列演算） | 確率モデルあり |
| 誤差の扱い | 全分散を説明しようとする | 共通分散のみ説明する |
| 因子の数 | 寄与率で決める | 適合度・解釈性で決める |
| 典型的な使用場面 | ML の前処理・可視化 | 心理尺度・アンケート分析 |

## 線形判別分析（LDA）

クラスを最もよく分離できる低次元空間を見つける手法です。**クラス間分散を最大化し、クラス内分散を最小化** する方向を探します。

PCA との違い：PCA はデータ全体の分散を最大化しますが、LDA は **ラベル情報を使って** クラス分離を最大化します。

```python
from sklearn.discriminant_analysis import LinearDiscriminantAnalysis
from sklearn.datasets import load_iris
from sklearn.model_selection import train_test_split
from sklearn.metrics import accuracy_score, classification_report

data = load_iris()
X, y = data.data, data.target
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.3, random_state=42)

lda = LinearDiscriminantAnalysis()
lda.fit(X_train, y_train)

# 分類器として使う
y_pred = lda.predict(X_test)
print(f"精度: {accuracy_score(y_test, y_pred):.3f}")
print(classification_report(y_test, y_pred, target_names=data.target_names))

# 次元削減としても使う（クラス数 - 1 次元まで削減可能）
X_lda = lda.transform(X)
plt.scatter(X_lda[:, 0], X_lda[:, 1], c=y, cmap='viridis', alpha=0.7)
plt.xlabel("LD1"); plt.ylabel("LD2")
plt.title("LDA による 2 次元投影")
plt.show()
```

### PCA vs LDA の使い分け

| 状況 | 推奨 |
|------|------|
| クラスラベルなし（教師なし） | PCA |
| クラスラベルあり・分類が目的 | LDA |
| ラベルはあるが分布の可視化が目的 | t-SNE / UMAP |
| 解釈可能な線形変換が必要 | PCA または LDA |

## クラスター分析

データをグループ（クラスター）に自動で分割する手法です。教師なし学習の一種です。

### k-means 法

最もシンプルなクラスタリング手法です。k 個のクラスター中心を繰り返し更新して収束させます。

```python
from sklearn.cluster import KMeans
from sklearn.datasets import make_blobs

# 人工データを生成
X, true_labels = make_blobs(n_samples=300, centers=4, cluster_std=0.8, random_state=42)

# エルボー法：最適な k の選択
inertias = []
K_range = range(1, 11)
for k in K_range:
    km = KMeans(n_clusters=k, random_state=42, n_init=10)
    km.fit(X)
    inertias.append(km.inertia_)

plt.plot(K_range, inertias, 'bo-')
plt.xlabel("k（クラスター数）"); plt.ylabel("慣性（クラスター内二乗和）")
plt.title("エルボー法：k の選択")
plt.show()
```

```python
# k=4 でクラスタリング
km = KMeans(n_clusters=4, random_state=42, n_init=10)
labels = km.fit_predict(X)

plt.scatter(X[:, 0], X[:, 1], c=labels, cmap='tab10', alpha=0.7)
plt.scatter(km.cluster_centers_[:, 0], km.cluster_centers_[:, 1],
            s=200, c='red', marker='X', zorder=5, label='中心')
plt.title("k-means クラスタリング結果")
plt.legend()
plt.show()
```

### 階層的クラスタリング

データを段階的に結合・分割し、デンドログラム（樹形図）で表示します。

```python
from scipy.cluster.hierarchy import linkage, dendrogram, fcluster
from sklearn.datasets import load_iris

data = load_iris()
X_scaled = StandardScaler().fit_transform(data.data)

# Ward 法（クラスター内の分散増加が最小になるよう結合）
Z = linkage(X_scaled, method='ward')

plt.figure(figsize=(12, 5))
dendrogram(Z, labels=data.target, leaf_rotation=90, leaf_font_size=8)
plt.title("階層的クラスタリング（Ward 法）")
plt.xlabel("サンプル"); plt.ylabel("距離")
plt.show()

# デンドログラムを見て切断位置を決め、ラベルを取得
labels = fcluster(Z, t=7.0, criterion='distance')
print("クラスター数:", len(np.unique(labels)))
```

## 正準相関分析（CCA）

2 つの変数群（X の集合と Y の集合）の間の **最大の相関関係** を見つける手法です。例：心理検査のスコア群と行動指標群の関係。

```python
from sklearn.cross_decomposition import CCA
from sklearn.datasets import load_linnerud

# Linnerud データ：運動能力(X) × 生理指標(Y)
data = load_linnerud()
X, Y = data.data, data.target

cca = CCA(n_components=2)
X_c, Y_c = cca.fit_transform(X, Y)

# 第 1・第 2 正準変量の相関
r1 = np.corrcoef(X_c[:, 0], Y_c[:, 0])[0, 1]
r2 = np.corrcoef(X_c[:, 1], Y_c[:, 1])[0, 1]
print(f"第 1 正準相関係数: {r1:.3f}")
print(f"第 2 正準相関係数: {r2:.3f}")
```

## 多次元尺度構成法（MDS）

変数間の距離・類似度行列から、低次元空間でのマップを作成します。地図のない距離表から地図を再現するイメージです。

```python
from sklearn.manifold import MDS
from sklearn.datasets import load_digits

# 手書き数字（64 次元）を 2 次元にマッピング
data = load_digits(n_class=6)
X_mds = MDS(n_components=2, random_state=42).fit_transform(data.data[:200])

plt.scatter(X_mds[:, 0], X_mds[:, 1], c=data.target[:200], cmap='tab10', alpha=0.7)
plt.colorbar(label='数字')
plt.title("MDS による手書き数字の 2 次元マップ")
plt.show()
```

## 対応分析（Correspondence Analysis）

カテゴリ変数のクロス表（分割表）を低次元空間に可視化する手法です。**行カテゴリと列カテゴリの関係**を同一の散布図上に表示できるのが特徴です。アンケート分析やテキストデータのカテゴリ分析で使います。

```
分割表の例：ブランド × 購買層
         若年層  中年層  高齢層
ブランド A   80     30     10
ブランド B   20     60     40
ブランド C   10     20     70
```

```python
import numpy as np
import pandas as pd
import matplotlib.pyplot as plt

# 対応分析は prince ライブラリを使うのが簡単
# pip install prince
import prince

# 分割表（ブランド × 購買層）
data = pd.DataFrame(
    [[80, 30, 10],
     [20, 60, 40],
     [10, 20, 70]],
    index=["ブランド A", "ブランド B", "ブランド C"],
    columns=["若年層", "中年層", "高齢層"]
)

ca = prince.CA(n_components=2, random_state=42)
ca = ca.fit(data)

# 行（ブランド）と列（購買層）を同一空間にプロット
ax = ca.plot_coordinates(X=data, figsize=(8, 5))
plt.title("対応分析：ブランド × 購買層")
plt.show()

# 各次元の寄与率
print(ca.eigenvalues_summary)
```

対応分析の内部では特異値分解（SVD）を使います：

$$
A = U \Sigma V^\top
$$

scipy だけで手動実装する場合：

```python
from scipy.linalg import svd

# 分割表を numpy 配列として準備
N = np.array([[80, 30, 10],
              [20, 60, 40],
              [10, 20, 70]], dtype=float)

n = N.sum()
P = N / n                           # 対応行列（確率化）
r = P.sum(axis=1, keepdims=True)    # 行マージン
c = P.sum(axis=0, keepdims=True)    # 列マージン

# 標準化残差行列
S = (P - r @ c) / np.sqrt(r @ c)

# SVD で主座標を取得
U, D, Vt = svd(S, full_matrices=False)

row_coords = np.diag(1 / r.ravel()**0.5) @ U[:, :2] * D[:2]
col_coords = np.diag(1 / c.ravel()**0.5) @ Vt[:2, :].T * D[:2]

# 寄与率
total_inertia = (D**2).sum()
print("次元ごとの寄与率:", (D[:2]**2 / total_inertia).round(3))
```

### MDS との違い

| 観点 | MDS | 対応分析 |
|------|-----|---------|
| 入力 | 距離・類似度行列 | 分割表（度数） |
| 対象 | 量的・質的データ | カテゴリ変数 |
| プロット | 行のみ | 行と列を同時にプロット |
| 用途 | 距離の保存 | カテゴリ間の関連の可視化 |

## 手法選択ガイド

| やりたいこと | 手法 | ラベルが必要か |
|------------|------|:------:|
| 次元削減・可視化 | PCA | 不要 |
| クラス分離の最大化 | LDA | 必要 |
| 潜在因子の解釈 | 因子分析 | 不要 |
| グループ分け（k を決める） | k-means | 不要 |
| グループの階層構造を見る | 階層的クラスタリング | 不要 |
| 2 変数群の関係を見る | 正準相関分析 | 不要 |
| 距離行列から地図を作る | MDS | 不要 |
| カテゴリ変数のクロス表を可視化 | 対応分析 | 不要 |
| 非線形な低次元化 | t-SNE / UMAP | 不要 |
