フーリエ解析¶

任意の信号（波）を周波数成分の重ね合わせとして分解する手法です。「音楽は複数の音（周波数）の合成」「株価の変動には日次・週次・年次の周期が混在」——フーリエ解析はこの「どんな周波数が混ざっているか」を明らかにします。

はじめて読む人へ¶

音声認識・音楽生成・時系列予測・MRI 画像処理——現代の ML で使われる多くの特徴量はフーリエ変換を経由しています。スペクトログラム（音声の時間-周波数マップ）は CNN や Transformer に入力される前にフーリエ変換で作られます。

読む前に押さえること¶

  • 微分・最適化基礎 の三角関数（sin・cos）の基本
  • 複素数 \(e^{i\theta} = \cos\theta + i\sin\theta\)（オイラーの公式）を知っていると数式の理解が速まりますが必須ではありません

読み終えたら説明できること¶

  • フーリエ変換が「時間領域 → 周波数領域」への変換であることを説明できる
  • DFT と FFT の違いを説明できる
  • Python で音声や時系列データのスペクトラム解析ができる

直感：音楽とフーリエ¶

ド（261 Hz）+ ミ（330 Hz）+ ソ（392 Hz）を同時に弾く ↓ 合成 複雑な波形（ドミソのコード）

フーリエ変換 ↓ 分解 261 Hz の成分：強 330 Hz の成分：強 392 Hz の成分：強 それ以外：ほぼゼロ フーリエ変換は「複雑な波 → それを作っている周波数の一覧」への変換です。

フーリエ級数¶

周期 \(T\) の関数は sin と cos の無限和で表せます。

\[ f(t) = \frac{a_0}{2} + \sum_{n=1}^{\infty} \left[ a_n \cos\!\left(\frac{2\pi n}{T} t\right) + b_n \sin\!\left(\frac{2\pi n}{T} t\right) \right] \]

係数 \(a_n\), \(b_n\) が「各周波数の強さ」を表します。

import numpy as np
import matplotlib.pyplot as plt

t = np.linspace(0, 2*np.pi, 1000)

# 方形波をフーリエ級数で近似（奇数次の sin の和）
def square_wave_approx(t, N):
    result = np.zeros_like(t)
    for n in range(1, N+1, 2):   # 1, 3, 5, ...
        result += (4 / (np.pi * n)) * np.sin(n * t)
    return result

fig, axes = plt.subplots(1, 3, figsize=(15, 4))
for ax, N in zip(axes, [1, 5, 50]):
    ax.plot(t, square_wave_approx(t, N))
    ax.set_title(f"N={N} 項")
    ax.set_ylim(-1.5, 1.5)
plt.suptitle("フーリエ級数で方形波を近似")
plt.tight_layout()
plt.savefig("fourier_series.png", dpi=150)

連続フーリエ変換¶

非周期関数にも適用できる変換です。

\[ \hat{f}(\xi) = \int_{-\infty}^{\infty} f(t)\, e^{-2\pi i \xi t}\, dt \]
  • \(f(t)\)：時間領域の信号
  • \(\hat{f}(\xi)\)：周波数 \(\xi\) の成分の強さ（複素数）
  • \(|\hat{f}(\xi)|\)：パワースペクトル（各周波数のエネルギー）

離散フーリエ変換（DFT）¶

コンピュータで扱うのは有限個の離散データです。\(N\) 個のデータ \(x[0], \ldots, x[N-1]\) に対して：

\[ X[k] = \sum_{n=0}^{N-1} x[n]\, e^{-2\pi i k n / N}, \quad k = 0, 1, \ldots, N-1 \]
import numpy as np
import matplotlib.pyplot as plt

# 3 Hz と 7 Hz の正弦波を合成
fs = 100         # サンプリング周波数 [Hz]
T = 1.0          # 信号の長さ [秒]
t = np.linspace(0, T, int(fs * T), endpoint=False)

signal = np.sin(2 * np.pi * 3 * t) + 0.5 * np.sin(2 * np.pi * 7 * t)

# DFT
X = np.fft.fft(signal)
freqs = np.fft.fftfreq(len(signal), 1/fs)

# パワースペクトル（片側）
N = len(signal)
power = (2 / N) * np.abs(X[:N//2])
freq_axis = freqs[:N//2]

plt.figure(figsize=(12, 4))
plt.subplot(1, 2, 1)
plt.plot(t, signal)
plt.title("時間領域の信号")
plt.xlabel("時間 [s]")

plt.subplot(1, 2, 2)
plt.plot(freq_axis, power)
plt.title("周波数スペクトル")
plt.xlabel("周波数 [Hz]")
plt.tight_layout()
plt.savefig("dft.png", dpi=150)
# → 3 Hz と 7 Hz のところだけスパイクが立つ

高速フーリエ変換（FFT）¶

DFT の計算量は \(O(N^2)\) ですが、FFT アルゴリズムで \(O(N \log N)\) に高速化されます。

手法 計算量 N=1024 での演算回数
DFT（素朴） \(O(N^2)\) 約 100 万
FFT \(O(N \log N)\) 約 1 万

numpy.fft.fft は自動的に FFT アルゴリズムを使います。

スペクトログラム（短時間フーリエ変換 STFT）¶

音声のように「周波数が時間とともに変化する」信号には、窓をずらしながら FFT を繰り返す STFT を使います。

import librosa
import librosa.display
import matplotlib.pyplot as plt
import numpy as np

# 音声ファイルの読み込み（librosa: pip install librosa）
# y, sr = librosa.load("speech.wav", sr=22050)

# デモ用にチャープ信号（周波数が時間とともに増加）を生成
sr = 22050
t = np.linspace(0, 3, sr * 3)
y = np.sin(2 * np.pi * (100 + 1000 * t / 3) * t).astype(np.float32)

# STFT でスペクトログラムを計算
D = librosa.stft(y, n_fft=2048, hop_length=512)
S_db = librosa.amplitude_to_db(np.abs(D), ref=np.max)

plt.figure(figsize=(10, 4))
librosa.display.specshow(S_db, sr=sr, hop_length=512,
                          x_axis="time", y_axis="log")
plt.colorbar(format="%+2.0f dB")
plt.title("スペクトログラム（STFT）")
plt.tight_layout()
plt.savefig("spectrogram.png", dpi=150)

ML での使われ方¶

音声データ → STFT → スペクトログラム（2D 画像） ↓ CNN / Transformer で処理 ↓ 音声認識・感情分類・音楽ジャンル分類

2 次元 FFT（画像処理）¶

画像の空間周波数を分析します。高周波 = エッジ・細部、低周波 = 全体的な明るさ。

from PIL import Image
import numpy as np
import matplotlib.pyplot as plt

# グレースケール画像の 2D FFT
img = np.random.randint(0, 255, (256, 256), dtype=np.uint8)  # ダミー画像
F = np.fft.fft2(img)
F_shift = np.fft.fftshift(F)   # 低周波を中心に移動
magnitude = np.log1p(np.abs(F_shift))

plt.subplot(1, 2, 1); plt.imshow(img, cmap="gray"); plt.title("元画像")
plt.subplot(1, 2, 2); plt.imshow(magnitude, cmap="hot"); plt.title("周波数スペクトラム")
plt.tight_layout()
plt.savefig("fft2d.png", dpi=150)

時系列分析への応用¶

import pandas as pd
import numpy as np
import matplotlib.pyplot as plt

# 月次売上データを想定（トレンド + 12ヶ月周期 + ノイズ）
np.random.seed(42)
n = 120  # 10年分の月次データ
t = np.arange(n)
sales = 100 + 0.5*t + 30*np.sin(2*np.pi*t/12) + np.random.normal(0, 5, n)

# FFT で周期成分を特定
X = np.fft.fft(sales)
freqs = np.fft.fftfreq(n, d=1)  # d=1: 月次
power = np.abs(X[:n//2])

# 強いピークの周期を表示
peak_idx = np.argsort(power)[::-1][1:4]  # DC成分を除いた上位3
for idx in peak_idx:
    if freqs[idx] > 0:
        period = 1 / freqs[idx]
        print(f"強い周期成分: {period:.1f} ヶ月（{power[idx]:.0f}）")
# → 12.0 ヶ月（最強）が検出される

数学的導出¶

フーリエ係数の導出（直交性の利用）¶

周期 \(T\) の関数 \(f(t)\) がフーリエ級数

\[ f(t) = \frac{a_0}{2} + \sum_{n=1}^{\infty} \left[ a_n \cos\!\left(\frac{2\pi n}{T} t\right) + b_n \sin\!\left(\frac{2\pi n}{T} t\right) \right] \]

で表されるとして、係数 \(a_n\)、\(b_n\) を求めます。

鍵となる直交性¶

三角関数は互いに直交します。すなわち、\(m \neq n\) のとき：

\[ \int_0^T \cos\!\left(\frac{2\pi m}{T}t\right) \cos\!\left(\frac{2\pi n}{T}t\right) dt = 0 \]
\[ \int_0^T \sin\!\left(\frac{2\pi m}{T}t\right) \sin\!\left(\frac{2\pi n}{T}t\right) dt = 0 \]
\[ \int_0^T \cos\!\left(\frac{2\pi m}{T}t\right) \sin\!\left(\frac{2\pi n}{T}t\right) dt = 0 \quad (\text{任意の } m, n) \]

\(m = n\) のとき：

\[ \int_0^T \cos^2\!\left(\frac{2\pi n}{T}t\right) dt = \int_0^T \sin^2\!\left(\frac{2\pi n}{T}t\right) dt = \frac{T}{2} \]

係数 \(a_n\) の導出（\(n \geq 1\)）¶

フーリエ級数の両辺に \(\cos\!\left(\frac{2\pi m}{T}t\right)\) を掛けて \([0, T]\) で積分します：

\[ \int_0^T f(t) \cos\!\left(\frac{2\pi m}{T}t\right) dt = \int_0^T \sum_n \left[ a_n \cos\!\left(\frac{2\pi n}{T}t\right) + b_n \sin\!\left(\frac{2\pi n}{T}t\right) \right] \cos\!\left(\frac{2\pi m}{T}t\right) dt \]

直交性により \(n \neq m\) の項はすべてゼロになり、\(n = m\) の項だけが残ります：

\[ = a_m \cdot \frac{T}{2} \]

したがって：

\[ \boxed{a_n = \frac{2}{T} \int_0^T f(t) \cos\!\left(\frac{2\pi n}{T} t\right) dt} \]

同様にして：

\[ \boxed{b_n = \frac{2}{T} \int_0^T f(t) \sin\!\left(\frac{2\pi n}{T} t\right) dt} \]

\(a_0 = \frac{2}{T}\int_0^T f(t)\,dt\) は \(f(t)\) の平均値の 2 倍です。

直感： フーリエ係数は「\(f(t)\) と各周波数成分の内積」です。\(f(t)\) が \(n\) 周波数の正弦波に似ているほど、その係数が大きくなります。これは線形代数の「成分を求めるには基底ベクトルとの内積を取る」操作と全く同じ構造です。

import numpy as np

def fourier_coeff(f_vals, t, T, n):
    """数値積分でフーリエ係数を計算"""
    dt = t[1] - t[0]
    a_n = (2/T) * np.trapz(f_vals * np.cos(2*np.pi*n*t/T), t)
    b_n = (2/T) * np.trapz(f_vals * np.sin(2*np.pi*n*t/T), t)
    return a_n, b_n

T = 2 * np.pi
t = np.linspace(0, T, 1000, endpoint=False)

# 方形波の係数
square = np.sign(np.sin(t))

for n in range(1, 6):
    a, b = fourier_coeff(square, t, T, n)
    print(f"n={n}: a_{n}={a:.4f}, b_{n}={b:.4f}")
# 奇数 n でのみ b_n が非ゼロ（4/(nπ)）になることを確認

DFT と連続フーリエ変換の関係¶

連続フーリエ変換：

\[ \hat{f}(\xi) = \int_{-\infty}^{\infty} f(t) e^{-2\pi i \xi t} dt \]

\(N\) 点の離散サンプル \(f_k = f(k/N)\) に対して、和で近似すると DFT：

\[ F_n = \sum_{k=0}^{N-1} f_k \cdot e^{-2\pi i kn/N} \]

オイラーの公式 \(e^{i\theta} = \cos\theta + i\sin\theta\) を代入すれば、フーリエ級数の cos・sin 係数を複素数 1 本で表したものが DFT であることが分かります。

確認問題¶

  1. フーリエ係数 \(a_n = \frac{2}{T}\int_0^T f(t)\cos(\frac{2\pi n}{T}t)\,dt\) が「内積」として解釈できる理由を、直交性の条件を使って説明してください。
  2. DFT の計算量は \(O(N^2)\)、FFT は \(O(N \log N)\) です。\(N = 1024\) のとき、何倍速くなりますか？
