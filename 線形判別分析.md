# 線形判別分析（LDA）

クラスを最もよく分離できる低次元空間を見つける手法です。**クラス間分散を最大化し、クラス内分散を最小化** する方向を探します。分類器としても、次元削減ツールとしても使えます。

---

## はじめて読む人へ

PCA が「データ全体のばらつきが最大の方向を探す」のに対し、LDA は「クラス同士が最もよく分かれる方向を探す」手法です。クラスラベルを知っているので、PCA より目的に沿った圧縮ができます。「猫と犬を最もよく区別できる 1 本の軸を引く」イメージです。

### 読み終えたら説明できること

- PCA と LDA の「最大化する量」の違いを説明できる
- Fisher 基準（クラス間分散 / クラス内分散）を数式で説明できる
- LDA が削減できる次元の上限（クラス数 - 1）を説明できる

## Fisher 基準の導出

### 問題設定

$K$ クラス、$p$ 次元のデータを $d$ 次元（$d \leq K-1$）に線形射影する行列 $W \in \mathbb{R}^{p \times d}$ を求めたい。

「良い射影」とは何か？ → **クラス間が広がり、クラス内が密集している** 射影です。

### クラス間散布行列 $S_B$

各クラスの重心が全体重心からどれだけ離れているかを測ります。

$$
S_B = \sum_{k=1}^{K} n_k (\mu_k - \mu)(\mu_k - \mu)^\top
$$

- $n_k$：クラス $k$ のサンプル数
- $\mu_k$：クラス $k$ の平均ベクトル
- $\mu$：全体平均ベクトル

### クラス内散布行列 $S_W$

各クラス内でサンプルが重心からどれだけばらついているかを測ります。

$$
S_W = \sum_{k=1}^{K} \sum_{i \in C_k} (x_i - \mu_k)(x_i - \mu_k)^\top
$$

### Fisher 基準

クラス間分散をクラス内分散で割った量を最大化する投影を求めます。

$$
J(W) = \frac{|W^\top S_B W|}{|W^\top S_W W|} \to \text{maximize}
$$

### 最適解の導出（一般化固有値問題）

$J(W)$ を最大化する $W$ は、次の一般化固有値問題の上位固有ベクトルです。

$$
S_B \mathbf{w} = \lambda S_W \mathbf{w}
$$

両辺を $S_W$ で割り（$S_W$ が正則の場合）、

$$
S_W^{-1} S_B \mathbf{w} = \lambda \mathbf{w}
$$

この通常の固有値問題を解くことで、Fisher 基準を最大化する投影方向 $\mathbf{w}$ が求まります。

> **なぜ固有値問題になるのか**  
> $\mathbf{w}$ 方向に射影したときの Fisher 基準は：
> $$J(\mathbf{w}) = \frac{\mathbf{w}^\top S_B \mathbf{w}}{\mathbf{w}^\top S_W \mathbf{w}}$$
> これは Rayleigh 商の形です。Rayleigh 商 $\frac{\mathbf{w}^\top A \mathbf{w}}{\mathbf{w}^\top B \mathbf{w}}$ の最大値は $B^{-1}A$ の最大固有値で与えられ、最大値を達成するベクトルが対応する固有ベクトルです。

### 次元削減上限

- $S_B$ のランクは最大 $K-1$（$K$ 個のクラス重心が存在し、全体重心を除くと独立なものは $K-1$ 個）
- したがって **LDA は最大 $K-1$ 次元** にしか削減できない

## 実装

```python
from sklearn.discriminant_analysis import LinearDiscriminantAnalysis
from sklearn.datasets import load_iris
from sklearn.model_selection import train_test_split
from sklearn.metrics import accuracy_score, classification_report
import matplotlib.pyplot as plt
import numpy as np

data = load_iris()
X, y = data.data, data.target
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.3, random_state=42)

lda = LinearDiscriminantAnalysis()
lda.fit(X_train, y_train)

# 分類器として使う
y_pred = lda.predict(X_test)
print(f"精度: {accuracy_score(y_test, y_pred):.3f}")
print(classification_report(y_test, y_pred, target_names=data.target_names))

# 次元削減としても使う（クラス数 - 1 = 2 次元まで削減可能）
X_lda = lda.transform(X)
plt.scatter(X_lda[:, 0], X_lda[:, 1], c=y, cmap='viridis', alpha=0.7)
plt.xlabel("LD1"); plt.ylabel("LD2")
plt.title("LDA による 2 次元投影")
plt.show()
```

### 散布行列を手計算して確認

```python
import numpy as np
from sklearn.datasets import load_iris
from sklearn.preprocessing import LabelEncoder

data = load_iris()
X, y = data.data, data.target
n_classes = len(np.unique(y))
n_features = X.shape[1]

# 全体平均
mu = X.mean(axis=0)

# クラス間散布行列 S_B
S_B = np.zeros((n_features, n_features))
for k in range(n_classes):
    Xk = X[y == k]
    mu_k = Xk.mean(axis=0)
    diff = (mu_k - mu).reshape(-1, 1)
    S_B += len(Xk) * (diff @ diff.T)

# クラス内散布行列 S_W
S_W = np.zeros((n_features, n_features))
for k in range(n_classes):
    Xk = X[y == k]
    mu_k = Xk.mean(axis=0)
    diff = Xk - mu_k
    S_W += diff.T @ diff

# 一般化固有値問題 S_W^{-1} S_B w = λ w を解く
eigenvalues, eigenvectors = np.linalg.eig(np.linalg.inv(S_W) @ S_B)

# 固有値の大きい順に並べ替え
idx = np.argsort(eigenvalues)[::-1]
eigenvalues = eigenvalues[idx].real
eigenvectors = eigenvectors[:, idx].real

print("固有値:", eigenvalues.round(3))
print("有効な次元数:", np.sum(eigenvalues > 1e-10))  # ランク(S_B) = K-1 = 2
```

## PCA vs LDA の使い分け

| 状況 | 推奨 |
|------|------|
| クラスラベルなし（教師なし） | PCA |
| クラスラベルあり・分類が目的 | LDA |
| ラベルはあるが分布の可視化が目的 | t-SNE / UMAP |
| 解釈可能な線形変換が必要 | PCA または LDA |
| クラス数が多い（$K > p$） | LDA は次元上限がある点に注意 |
