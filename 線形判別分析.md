# 線形判別分析（LDA）

クラスを最もよく分離できる低次元空間を見つける手法です。**クラス間分散を最大化し、クラス内分散を最小化** する方向を探します。分類器としても、次元削減ツールとしても使えます。

PCA との違い：PCA はデータ全体の分散を最大化しますが、LDA は **ラベル情報を使って** クラス分離を最大化します。

---

## 実装

```python
from sklearn.discriminant_analysis import LinearDiscriminantAnalysis
from sklearn.datasets import load_iris
from sklearn.model_selection import train_test_split
from sklearn.metrics import accuracy_score, classification_report
import matplotlib.pyplot as plt

data = load_iris()
X, y = data.data, data.target
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.3, random_state=42)

lda = LinearDiscriminantAnalysis()
lda.fit(X_train, y_train)

# 分類器として使う
y_pred = lda.predict(X_test)
print(f"精度: {accuracy_score(y_test, y_pred):.3f}")
print(classification_report(y_test, y_pred, target_names=data.target_names))

# 次元削減としても使う（クラス数 - 1 次元まで削減可能）
X_lda = lda.transform(X)
plt.scatter(X_lda[:, 0], X_lda[:, 1], c=y, cmap='viridis', alpha=0.7)
plt.xlabel("LD1"); plt.ylabel("LD2")
plt.title("LDA による 2 次元投影")
plt.show()
```

---

## PCA vs LDA の使い分け

| 状況 | 推奨 |
|------|------|
| クラスラベルなし（教師なし） | PCA |
| クラスラベルあり・分類が目的 | LDA |
| ラベルはあるが分布の可視化が目的 | t-SNE / UMAP |
| 解釈可能な線形変換が必要 | PCA または LDA |
