# 教師なし学習

**正解ラベルなしでデータの中のパターン・構造を見つける**手法です。「似たデータをまとめる（クラスタリング）」「複雑なデータをシンプルに圧縮する（次元削減）」「普通とは違うデータを見つける（異常検知）」の 3 つが代表的な使い方です。

---

## はじめて読む人へ

教師あり学習は「正解ラベル付きデータ」が必要でしたが、現実のデータには正解が付いていないことがほとんどです。「このお客さんたちはどんなグループに分けられるか」「このデータのどこが重要な情報か」という問いに答えるのが教師なし学習です。

### 読む前に押さえること

- [教師あり学習](教師あり学習.md) との違いは「正解データがあるかどうか」
- 結果は「答え」ではなく「パターン」として解釈が必要

### 読み終えたら説明できること

- クラスタリング・次元削減・異常検知それぞれの目的を説明できる
- どんなデータ・問題に教師なし学習を使うかを説明できる

---

## 教師あり vs 教師なし学習

| 教師あり学習 | 教師なし学習 |
|---|---|
| 入力: [メールの文章]  → 正解ラベル: [スパム/通常] | 入力: [1000人の購買データ]  → 正解ラベル: なし |
| 目的: 正解を当てるパターンを学ぶ | 目的: 自然なグループ・構造を発見する |
---

## 3 つの種類

### クラスタリング（グループ分け）

似たデータを自動でまとめます。何グループあるかを事前に決める手法と、データの密度から自動で決まる手法があります。

例: 1000 人の顧客を購買パターンで分ける

クラスタ 1: 「週 1 回・少額購入」 → ライトユーザー
クラスタ 2: 「月 1 回・高額購入」 → 高単価ユーザー
クラスタ 3: 「頻繁・幅広い品目」 → ヘビーユーザー
| 手法 | 特徴 | 詳細 |
|------|------|------|
| k-means | k を指定して球形のグループに分割 | [クラスター分析](クラスター分析.md) |
| DBSCAN | 密度で自動にグループ数を決定・外れ値検出 | [クラスター分析](クラスター分析.md) |
| 階層的クラスタリング | k を決めずにデンドログラムで可視化 | [クラスター分析](クラスター分析.md) |

### 次元削減（圧縮・可視化）

たくさんの特徴量（列）を持つデータを、情報を保ちながら少ない次元に圧縮します。100 個の特徴量を 2 個に圧縮して散布図で見ると、データの構造が見えてきます。

例: 50 種類の成分を測った食品データ
→ PCA で 2 次元に圧縮
→ 散布図で「肉類のグループ」「野菜類のグループ」が見える
| 手法 | 特徴 | 詳細 |
|------|------|------|
| PCA（主成分分析） | 分散が最大の方向に圧縮（線形）| [主成分分析（PCA）](主成分分析.md) |
| t-SNE | 近くにある点を近くに配置（非線形・可視化向き）| [主成分分析（PCA）](主成分分析.md) |
| UMAP | t-SNE より高速・大規模データ向き | [主成分分析（PCA）](主成分分析.md) |
| 因子分析 | 潜在的な共通因子を抽出 | [因子分析](因子分析.md) |

### 異常検知（外れ値の発見）

「正常なデータ」で学習し、それと大きく違うデータを異常として検出します。不正検知・機械の故障予測・医療データの異常などに使います。

例: 工場センサーのデータ
→ 通常時のパターンを学習
→ 異常な動きをしたとき「異常値スコア」が高くなる → アラート
```python
from sklearn.ensemble import IsolationForest

# 正常データで学習して異常スコアを計算
iso = IsolationForest(contamination=0.05, random_state=42)
labels = iso.fit_predict(X_scaled)
# labels == -1 → 異常、labels == 1 → 正常
```

## どのときに使うか

| 状況 | 向いている手法 |
|------|-------------|
| 顧客・データをグループ分けしたい | クラスタリング |
| 特徴量が多くてモデルが遅い / 可視化したい | PCA などの次元削減 |
| 外れ値・不正なデータを検出したい | 異常検知 |
| 次元削減してから機械学習モデルに入力したい | PCA → 教師あり学習 |

## 確認問題

1. 教師あり学習と教師なし学習の違いを「正解ラベル」という言葉を使って説明してください。
2. クラスタリングの結果に「正解・不正解」がない理由を説明してください。
3. 100 次元のデータを 2 次元に圧縮するメリットとデメリットは何ですか？
