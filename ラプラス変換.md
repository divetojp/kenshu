# ラプラス変換・Z変換

微分方程式を「代数方程式」に変換する手法です。「時間領域での複雑な微分演算」が「$s$ 領域での掛け算・割り算」になるため、制御理論・信号処理・フィルタ設計で広く使われます。離散時間版は **Z 変換**です。

---

## はじめて読む人へ

「スプリングの振動」「RC 回路の充放電」「ロボットアームの制御」——これらはすべて微分方程式で記述されます。ラプラス変換を使うと、微分方程式を解く代わりに多項式の割り算だけで答えが出ます。デジタル信号処理・音声フィルタ・制御システムの設計に直結します。

### 読む前に押さえること

- [微分・最適化基礎](微分・最適化基礎.md) の微分・指数関数
- [フーリエ解析](フーリエ解析.md) を先に読むと、周波数領域変換の考え方が共通であることが分かります

### 読み終えたら説明できること

- ラプラス変換が「微分を $s$ の掛け算に変換する」ことを説明できる
- 伝達関数とは何かを説明できる
- Z 変換がラプラス変換の離散版であることを説明できる

---

## ラプラス変換の定義

時間関数 $f(t)$（$t \geq 0$）のラプラス変換：

$$
F(s) = \mathcal{L}\{f(t)\} = \int_0^{\infty} f(t)\, e^{-st}\, dt
$$

$s = \sigma + i\omega$ は複素数の変数です。フーリエ変換が純虚数軸（$s = i\omega$）上の評価なのに対し、ラプラス変換は複素平面全体を使うため、より広い関数クラスに適用できます。

---

## 基本的な変換ペア

| $f(t)$ | $F(s)$ | 条件 |
|--------|--------|------|
| $1$（単位ステップ）| $\dfrac{1}{s}$ | $\text{Re}(s) > 0$ |
| $t$ | $\dfrac{1}{s^2}$ | |
| $e^{at}$ | $\dfrac{1}{s - a}$ | $\text{Re}(s) > a$ |
| $\sin(\omega t)$ | $\dfrac{\omega}{s^2 + \omega^2}$ | |
| $\cos(\omega t)$ | $\dfrac{s}{s^2 + \omega^2}$ | |
| $\delta(t)$（インパルス）| $1$ | |

```python
import sympy as sp

t, s = sp.symbols('t s', positive=True)

# SymPy でラプラス変換
f1 = sp.exp(-2 * t)
F1 = sp.laplace_transform(f1, t, s, noconds=True)
print(f"L{{e^(-2t)}} = {F1}")    # 1/(s+2)

f2 = sp.sin(3 * t)
F2 = sp.laplace_transform(f2, t, s, noconds=True)
print(f"L{{sin(3t)}} = {F2}")    # 3/(s**2 + 9)
```

## 微分のラプラス変換（最重要）

$$
\mathcal{L}\{f'(t)\} = sF(s) - f(0)
$$

$$
\mathcal{L}\{f''(t)\} = s^2 F(s) - sf(0) - f'(0)
$$

**微分 → $s$ の掛け算** になります。これがラプラス変換最大の威力です。

### 例：RC 回路の微分方程式を解く

$$
RC \frac{dV_C}{dt} + V_C = V_{in}
$$

ラプラス変換すると（初期値 $V_C(0) = 0$）：

$$
RCs \cdot V_C(s) + V_C(s) = V_{in}(s)
$$

$$
V_C(s) = \frac{V_{in}(s)}{RCs + 1} = \frac{1}{RCs + 1} \cdot V_{in}(s)
$$

微分方程式が**代数方程式**になりました。$\frac{1}{RCs+1}$ が**伝達関数**です。

---

## 伝達関数とシステム解析

伝達関数 $H(s) = \frac{Y(s)}{X(s)}$ はシステムの入出力関係を $s$ 領域で表します。

```python
from scipy import signal
import numpy as np
import matplotlib.pyplot as plt

# RC 回路の伝達関数: H(s) = 1 / (RCs + 1)
R = 1000    # 1kΩ
C = 1e-6    # 1μF
tau = R * C  # 時定数

# 分子・分母の係数（降べきの順）
num = [1]           # 分子: 1
den = [tau, 1]      # 分母: τs + 1

sys = signal.TransferFunction(num, den)

# ステップ応答（単位ステップ入力に対する出力）
t, y = signal.step(sys)

plt.figure(figsize=(8, 4))
plt.plot(t * 1000, y)    # ms 単位で表示
plt.axhline(0.632, color='red', linestyle='--', label='63.2%（時定数 τ）')
plt.xlabel("時間 [ms]")
plt.ylabel("出力電圧（正規化）")
plt.title(f"RC 回路のステップ応答（τ = {tau*1000:.1f} ms）")
plt.legend()
plt.grid()
plt.savefig("rc_step.png", dpi=150)
```

### ボード線図（周波数特性）

```python
# 周波数応答
w, H = signal.freqs(num, den)

fig, (ax1, ax2) = plt.subplots(2, 1, figsize=(8, 6))

# ゲイン（dB）
ax1.semilogx(w, 20 * np.log10(np.abs(H)))
ax1.set_ylabel("ゲイン [dB]")
ax1.axvline(1/tau, color='red', linestyle='--', label=f'カットオフ周波数 fc={1/tau/(2*np.pi):.1f} Hz')
ax1.legend(); ax1.grid()

# 位相
ax2.semilogx(w, np.degrees(np.angle(H)))
ax2.set_xlabel("角周波数 [rad/s]"); ax2.set_ylabel("位相 [度]")
ax2.grid()

plt.suptitle("ボード線図（RC ローパスフィルタ）")
plt.tight_layout()
plt.savefig("bode.png", dpi=150)
```

## 逆ラプラス変換

$s$ 領域の $F(s)$ から時間領域の $f(t)$ に戻す操作です。部分分数分解が基本手法です。

$$
F(s) = \frac{1}{(s+1)(s+3)} = \frac{A}{s+1} + \frac{B}{s+3}
$$

```python
import sympy as sp

s = sp.Symbol('s')
F = 1 / ((s+1)*(s+3))

# 部分分数分解
partial = sp.apart(F, s)
print(partial)  # -1/(2*(s+3)) + 1/(2*(s+1))

# 逆ラプラス変換
t = sp.Symbol('t', positive=True)
f = sp.inverse_laplace_transform(F, s, t)
print(f)  # exp(-t)/2 - exp(-3t)/2
```

## Z 変換（離散ラプラス変換）

デジタル信号処理・デジタルフィルタ・差分方程式の解析に使います。離散時間信号 $x[n]$ の Z 変換：

$$
X(z) = \sum_{n=-\infty}^{\infty} x[n]\, z^{-n}
$$

$z = e^{sT}$（$T$：サンプリング周期）の関係でラプラス変換と繋がっています。

| $x[n]$ | $X(z)$ |
|--------|--------|
| $\delta[n]$（単位インパルス）| $1$ |
| $u[n]$（単位ステップ）| $\dfrac{z}{z-1}$ |
| $a^n u[n]$ | $\dfrac{z}{z-a}$ |

```python
# デジタルフィルタ設計（Z 変換の応用）
from scipy import signal
import numpy as np

# バターワースローパスフィルタ（カットオフ周波数 = ナイキスト周波数の 20%）
b, a = signal.butter(N=4, Wn=0.2, btype='low')

# フィルタの周波数特性
w, h = signal.freqz(b, a)

import matplotlib.pyplot as plt
plt.plot(w / np.pi, 20 * np.log10(np.abs(h)))
plt.xlabel("正規化周波数（× π rad/sample）")
plt.ylabel("ゲイン [dB]")
plt.title("4次バターワースローパスフィルタ")
plt.grid()
plt.savefig("butterworth.png", dpi=150)
```

## フーリエ変換・ラプラス変換・Z 変換の比較

| 変換 | 入力 | 変数 | 主な用途 |
|------|------|------|---------|
| フーリエ変換 | 連続・非周期 | $\omega$（実数周波数）| 周波数スペクトル分析 |
| ラプラス変換 | 連続・$t \geq 0$ | $s = \sigma + i\omega$ | 制御・回路解析 |
| DFT/FFT | 離散・有限長 | 離散周波数 | 数値スペクトル解析 |
| Z 変換 | 離散 | $z$（複素数）| デジタルフィルタ |

## 数学的導出

### 微分の変換 $\mathcal{L}\{f'(t)\} = sF(s) - f(0)$ の証明

ラプラス変換の定義：$F(s) = \int_0^\infty f(t) e^{-st} dt$ に対して、$f'(t)$ の変換を部分積分で求めます。

$$
\mathcal{L}\{f'(t)\} = \int_0^\infty f'(t) e^{-st} dt
$$

部分積分 $\int u\,dv = [uv] - \int v\,du$（$u = e^{-st}$、$dv = f'(t)dt$ とおく）：

$$
= \left[f(t) e^{-st}\right]_0^\infty - \int_0^\infty f(t)(-s)e^{-st} dt
$$

$\text{Re}(s) > 0$ なら $f(t)e^{-st} \to 0$（$t \to \infty$）として：

$$
= 0 - f(0) + s \int_0^\infty f(t) e^{-st} dt
$$

$$
\boxed{\mathcal{L}\{f'(t)\} = sF(s) - f(0)}
$$

**高階微分への拡張：**  
$f''(t)$ には繰り返し適用して $s^2 F(s) - sf(0) - f'(0)$。初期条件が自然に組み込まれます。

---

### 1 階 ODE の完全な解法例

$$
\tau \frac{dy}{dt} + y = u(t), \quad y(0) = 0
$$

両辺をラプラス変換：

$$
\tau [sY(s) - y(0)] + Y(s) = U(s)
$$

$y(0) = 0$ を代入して $Y(s)$ について解くと：

$$
(\tau s + 1)Y(s) = U(s) \quad \Rightarrow \quad Y(s) = \underbrace{\frac{1}{\tau s + 1}}_{H(s)} U(s)
$$

この $H(s) = \frac{1}{\tau s + 1}$ が **伝達関数**（入力と出力の比）です。

**ステップ応答（$u(t) = 1$, $U(s) = 1/s$）：**

$$
Y(s) = \frac{1}{s(\tau s + 1)} = \frac{1}{s} - \frac{\tau}{\tau s + 1}
$$

逆変換して：

$$
y(t) = 1 - e^{-t/\tau}
$$

時定数 $\tau$ が「入力に対して出力が $e^{-1} \approx 63\%$ 追従するまでの時間」であることが式から直接読み取れます。

---

### 伝達関数とフーリエ変換の接続

$s = i\omega$（$\sigma = 0$、純虚数）と置くとラプラス変換はフーリエ変換に一致します：

$$
H(i\omega) = \frac{1}{\tau (i\omega) + 1} = \frac{1}{1 + i\omega\tau}
$$

- $|H(i\omega)| = \frac{1}{\sqrt{1 + (\omega\tau)^2}}$：**振幅特性**（ボード線図の縦軸）
- $\angle H(i\omega) = -\arctan(\omega\tau)$：**位相特性**

$\omega \gg 1/\tau$（高周波）では $|H| \to 0$ → **ローパスフィルタ** として動作することが読み取れます。
