# 多次元尺度構成法（MDS）

変数間の **距離・類似度行列** から、低次元空間でのマップを作成します。「地図のない距離表から地図を再現する」イメージです。都市間の距離データから地図を再構成する例が有名です。

---

## はじめて読む人へ

「東京・大阪・福岡・札幌の距離一覧表だけから、地図を再現できるか」——MDS はまさにこれをやります。距離や類似度さえ分かれば、元のデータが何次元であっても 2D・3D で位置関係を可視化できます。PCA は元の変数から出発しますが、MDS は「距離・類似度」から出発するのが大きな違いです。

### 読む前に押さえること

- [主成分分析（PCA）](主成分分析.md) — 次元削減の基本的な考え方
- [クラスター分析](クラスター分析.md) — 距離に基づくグルーピングとの対比

### 読み終えたら説明できること

- MDS が「距離行列を入力とする」点で PCA と何が違うかを説明できる
- MDS・PCA・t-SNE の使い分けを説明できる
- どんなデータに MDS が向いているかを説明できる

---

## 仕組み

MDS は「なるべく元の距離関係を保ちながら低次元に配置する」手法です。

$$
\text{minimize} \sum_{i < j} \left( d_{ij} - \hat{d}_{ij} \right)^2
$$

- $d_{ij}$：元の距離（入力）
- $\hat{d}_{ij}$：低次元空間での距離

---

## 実装

```python
import matplotlib.pyplot as plt
from sklearn.manifold import MDS
from sklearn.datasets import load_digits

# 手書き数字（64 次元）を 2 次元にマッピング
data = load_digits(n_class=6)
X_mds = MDS(n_components=2, random_state=42).fit_transform(data.data[:200])

plt.scatter(X_mds[:, 0], X_mds[:, 1], c=data.target[:200], cmap='tab10', alpha=0.7)
plt.colorbar(label='数字')
plt.title("MDS による手書き数字の 2 次元マップ")
plt.show()
```

---

## MDS vs 他の次元削減手法

| 観点 | MDS | PCA | t-SNE |
|------|-----|-----|-------|
| 入力 | 距離・類似度行列 | データ行列 | データ行列 |
| 保持するもの | 距離関係 | 分散（大域構造） | 局所構造 |
| 線形性 | 線形（古典的 MDS） | 線形 | 非線形 |
| 用途 | 距離データの可視化 | 前処理・圧縮 | クラスター可視化 |

## 確認問題

1. MDS への入力は「距離行列」ですが、PCA への入力は何ですか？この違いがどんな場面で MDS を有利にしますか？
2. 元の距離関係を完全に保持できない（誤差が出る）とき、MDS は何を最小化しますか？
3. 「アンケートで聞いた商品間の類似度評価」から商品マップを作りたい場合、PCA ではなく MDS を使う理由を説明してください。
