# 多次元尺度構成法（MDS）

変数間の **距離・類似度行列** から、低次元空間でのマップを作成します。「地図のない距離表から地図を再現する」イメージです。都市間の距離データから地図を再構成する例が有名です。

---

## はじめて読む人へ

「東京・大阪・福岡・札幌の距離一覧表だけから、地図を再現できるか」——MDS はまさにこれをやります。距離や類似度さえ分かれば、元のデータが何次元であっても 2D・3D で位置関係を可視化できます。PCA は元の変数から出発しますが、MDS は「距離・類似度」から出発するのが大きな違いです。

## 古典的 MDS（Classical MDS）のアルゴリズム導出

### 入力：距離行列 $D$

$n$ 点の距離行列 $D \in \mathbb{R}^{n \times n}$（$d_{ij}$ = 点 $i$ と点 $j$ の距離）から座標行列 $X \in \mathbb{R}^{n \times d}$ を復元する問題です。

### STEP 1：距離を内積に変換

ユークリッド距離の公式 $d_{ij}^2 = \|x_i - x_j\|^2 = \|x_i\|^2 - 2x_i^\top x_j + \|x_j\|^2$ を利用します。

内積行列 $B$ を $B_{ij} = x_i^\top x_j$ とすると：

$$
d_{ij}^2 = B_{ii} - 2B_{ij} + B_{jj}
$$

### STEP 2：二重中心化（Double Centering）

重心が原点になる（$\sum_i x_i = 0$）という制約のもとで、$B$ を $D$ から復元します。

中心化行列 $H = I - \frac{1}{n}\mathbf{1}\mathbf{1}^\top$ を両側から掛けます：

$$
B = -\frac{1}{2} H D^{(2)} H
$$

ここで $D^{(2)}_{ij} = d_{ij}^2$（各要素を二乗した行列）です。

**なぜこれで B が求まるのか：**  
$H$ をかけることで行和・列和がゼロになり（= 重心が原点）、$d_{ij}^2$ を内積 $B_{ij}$ に変換できます。

### STEP 3：$B$ を固有値分解して座標を復元

$$
B = V \Lambda V^\top
$$

$\Lambda = \text{diag}(\lambda_1, \ldots, \lambda_n)$（固有値、降順）として、上位 $d$ 個の正の固有値を使って：

$$
X = V_d \Lambda_d^{1/2}
$$

これが求める低次元座標行列です。

> **なぜ正の固有値だけ使うのか：**  
> 距離が真のユークリッド距離であれば $B$ は半正定値（固有値 ≥ 0）になります。負の固有値があれば、その分は「ユークリッド空間では表現できないひずみ」を意味するため、正の固有値のみで近似します。

## Stress 関数（非計量 MDS）

### 古典的 MDS の限界

古典的 MDS はユークリッド距離を仮定します。実際のデータが「順位データ」（A より B の方が似ている、という情報だけ）の場合は別の方法が必要です。

### Stress の定義

非計量 MDS（SMACOF など）は、元の距離と低次元空間での距離の「ずれ」を最小化します：

$$
\text{Stress} = \sqrt{\frac{\sum_{i < j} (d_{ij} - \hat{d}_{ij})^2}{\sum_{i < j} d_{ij}^2}}
$$

- $d_{ij}$：元の距離（入力）
- $\hat{d}_{ij}$：低次元空間での距離

Stress = 0 なら完全再現、Stress = 0.05 以下なら「良好」、0.2 以上なら「悪い」の目安があります。

## 実装

```python
import matplotlib.pyplot as plt
from sklearn.manifold import MDS
from sklearn.datasets import load_digits
import numpy as np

# 手書き数字（64 次元）を 2 次元にマッピング
data = load_digits(n_class=6)
mds = MDS(n_components=2, random_state=42, normalized_stress='auto')
X_mds = mds.fit_transform(data.data[:200])

plt.scatter(X_mds[:, 0], X_mds[:, 1], c=data.target[:200], cmap='tab10', alpha=0.7)
plt.colorbar(label='数字')
plt.title(f"MDS による手書き数字の 2 次元マップ（Stress={mds.stress_:.4f}）")
plt.show()
```

### 古典的 MDS を手計算で実装

```python
import numpy as np

def classical_mds(D_squared, d=2):
    """距離行列の二乗 D_squared から d 次元座標を返す"""
    n = D_squared.shape[0]
    
    # 二重中心化
    H = np.eye(n) - np.ones((n, n)) / n
    B = -0.5 * H @ D_squared @ H
    
    # 固有値分解
    eigenvalues, eigenvectors = np.linalg.eigh(B)
    
    # 降順に並べ替え
    idx = np.argsort(eigenvalues)[::-1]
    eigenvalues = eigenvalues[idx]
    eigenvectors = eigenvectors[:, idx]
    
    # 正の固有値のみ使って座標を復元
    eigenvalues_d = np.maximum(eigenvalues[:d], 0)
    X = eigenvectors[:, :d] * np.sqrt(eigenvalues_d)
    return X

# 都市間距離の例
# 東京(0), 大阪(1), 福岡(2), 札幌(3) 単位:km
cities = ["東京", "大阪", "福岡", "札幌"]
D = np.array([
    [0,   550, 1090, 830],
    [550, 0,   550, 1365],
    [1090, 550, 0,  1870],
    [830, 1365, 1870, 0]
], dtype=float)

X = classical_mds(D**2, d=2)
print("2次元座標:")
for city, coord in zip(cities, X):
    print(f"  {city}: ({coord[0]:.0f}, {coord[1]:.0f})")
```

## MDS vs 他の次元削減手法

| 観点 | MDS | PCA | t-SNE |
|------|-----|-----|-------|
| 入力 | 距離・類似度行列 | データ行列 | データ行列 |
| 保持するもの | 距離関係 | 分散（大域構造） | 局所構造 |
| 線形性 | 線形（古典的 MDS） | 線形 | 非線形 |
| 用途 | 距離データの可視化 | 前処理・圧縮 | クラスター可視化 |
| 元データ不要 | **◎（距離だけでよい）** | × | × |
