## k-means 法

最もシンプルなクラスタリング手法です。k 個のクラスター中心を繰り返し更新して収束させます。

```python
from sklearn.cluster import KMeans
from sklearn.datasets import make_blobs
import matplotlib.pyplot as plt

# 人工データを生成
X, true_labels = make_blobs(n_samples=300, centers=4, cluster_std=0.8, random_state=42)

# エルボー法：最適な k の選択
inertias = []
K_range = range(1, 11)
for k in K_range:
    km = KMeans(n_clusters=k, random_state=42, n_init=10)
    km.fit(X)
    inertias.append(km.inertia_)

plt.plot(K_range, inertias, 'bo-')
plt.xlabel("k（クラスター数）"); plt.ylabel("慣性（クラスター内二乗和）")
plt.title("エルボー法：k の選択")
plt.show()
```

```python
# k=4 でクラスタリング
km = KMeans(n_clusters=4, random_state=42, n_init=10)
labels = km.fit_predict(X)

plt.scatter(X[:, 0], X[:, 1], c=labels, cmap='tab10', alpha=0.7)
plt.scatter(km.cluster_centers_[:, 0], km.cluster_centers_[:, 1],
            s=200, c='red', marker='X', zorder=5, label='中心')
plt.title("k-means クラスタリング結果")
plt.legend()
plt.show()
```

## 階層的クラスタリング

データを段階的に結合・分割し、デンドログラム（樹形図）で表示します。k を事前に決めなくてよいのが特徴です。

```python
import numpy as np
from scipy.cluster.hierarchy import linkage, dendrogram, fcluster
from sklearn.datasets import load_iris
from sklearn.preprocessing import StandardScaler

data = load_iris()
X_scaled = StandardScaler().fit_transform(data.data)

# Ward 法（クラスター内の分散増加が最小になるよう結合）
Z = linkage(X_scaled, method='ward')

plt.figure(figsize=(12, 5))
dendrogram(Z, labels=data.target, leaf_rotation=90, leaf_font_size=8)
plt.title("階層的クラスタリング（Ward 法）")
plt.xlabel("サンプル"); plt.ylabel("距離")
plt.show()

# デンドログラムを見て切断位置を決め、ラベルを取得
labels = fcluster(Z, t=7.0, criterion='distance')
print("クラスター数:", len(np.unique(labels)))
```

---

## k-means vs 階層的クラスタリング

| 観点 | k-means | 階層的クラスタリング |
|------|---------|-----------------|
| k の指定 | 事前に必要 | 不要（後から決められる） |
| 計算量 | $O(nkt)$（高速） | $O(n^2 \log n)$（低速） |
| データ規模 | 大規模 OK | 数千件以下が目安 |
| クラスター形状 | 球状を仮定 | 形状の仮定なし |
| 結果の再現性 | 初期値依存（乱数あり） | 決定的 |
