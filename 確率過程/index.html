確率過程¶

時間とともに変化する確率的な現象を数学的に扱うフレームワークです。「株価の変動」「ランダムウォーク」「ニューロンの発火」——これらは時刻ごとに確率変数が定まる確率過程です。機械学習では MCMC（ベイズ推論のサンプリング）・ガウス過程回帰・強化学習の MDP の基盤になります。

はじめて読む人へ¶

確率変数が「1 つの値」ではなく「時系列」として続く場面が確率過程です。ベイズ理論で学んだ MCMC は「マルコフ連鎖」を使ったサンプリング手法、ガウス過程回帰は「無限次元のガウス分布」——どちらもこのページの概念が土台です。

読む前に押さえること¶

  • 確率・統計基礎 の条件付き確率・確率変数・期待値
  • ベイズ理論 の事前・事後分布（MCMC との接続のため）

読み終えたら説明できること¶

  • マルコフ連鎖の「現在の状態だけで未来が決まる」性質を説明できる
  • MCMC が事後分布からのサンプリングに使われる仕組みを説明できる
  • ガウス過程が「関数の確率分布」であることを説明できる

確率過程とは¶

時刻 \(t\) において確率変数 \(X_t\) が定まる族 \(\{X_t\}_{t \geq 0}\)（または \(\{X_n\}_{n=0,1,2,\ldots}\)）です。

1 つの確率変数（例：\(X \sim N(0, 1)\)）は 1 つの数値をランダムに引くものです。確率過程はこれを時系列に拡張した概念で、各時刻に確率変数が定まります。例えば \(X_0 = 0,\; X_1 = 0.3,\; X_2 = -0.1,\; X_3 = 0.7, \ldots\) のように、各時刻で確率変数の値が確率的に変化していく無限次元の列です。

ランダムウォーク¶

最もシンプルな確率過程です。各ステップで +1 か -1 に等確率で動きます。

\[ X_n = X_0 + \sum_{k=1}^{n} Z_k, \quad Z_k \sim \text{Uniform}(\{-1, +1\}) \]

np.cumsum(np.random.choice([-1, 1], size=n)) でシミュレーションできます。

重要な性質：\(n\) ステップ後の標準偏差は \(\sqrt{n}\) に比例します（「拡散」）。これは独立な確率変数の分散の加算性から直接導かれます。金融の「ボラティリティが保有期間の平方根に比例する」という経験則の数学的根拠です。

マルコフ連鎖¶

現在の状態だけが次の状態を決める（過去は不要）性質をマルコフ性といいます。

\[ P(X_{n+1} = j \mid X_n = i, X_{n-1}, \ldots, X_0) = P(X_{n+1} = j \mid X_n = i) = P_{ij} \]

遷移行列¶

import numpy as np

# 天気の Markov 連鎖: 晴=0, 曇=1, 雨=2
P = np.array([
    [0.7, 0.2, 0.1],   # 晴 → 晴:70%, 曇:20%, 雨:10%
    [0.3, 0.4, 0.3],   # 曇 → 晴:30%, 曇:40%, 雨:30%
    [0.2, 0.3, 0.5],   # 雨 → 晴:20%, 曇:30%, 雨:50%
])

# 定常分布（長期的な各状態の確率）: π P = π を固有値で解く
eigenvalues, eigenvectors = np.linalg.eig(P.T)
stationary = eigenvectors[:, np.argmax(np.isclose(eigenvalues, 1))]
stationary = np.real(stationary / stationary.sum())
print(f"定常分布: {stationary.round(3)}")

n ステップ後の遷移行列は \(P^n\)（行列の n 乗）で得られます。十分なステップ後、遷移行列の各行は定常分布に収束します。

MCMC（マルコフ連鎖モンテカルロ）¶

直接サンプリングが難しい複雑な確率分布から、マルコフ連鎖を使ってサンプリングする手法です。ベイズ推論で事後分布 \(p(\theta | \text{data})\) からのサンプルを得るために使われます。

メトロポリス・ヘイスティングス法¶

アルゴリズム：

  1. 初期値 θ₀ を設定
  2. 各ステップで： a. 提案分布（例：正規分布）から候補 θ' を生成 b. 受理確率 α = min(1, p(θ'|data) / p(θ|data)) を計算 c. 確率 α で θ' を受理、1-α で棄却（現在の θ を維持）
  3. 十分なステップ後（バーンイン後）、サンプルの分布が p(θ|data) に収束 この手続きが事後分布に収束することは、詳細つり合い条件によって保証されます（後述）。pymc ライブラリでは NUTS（No-U-Turn Sampler）という高効率版が使われます。

受理率は通常 20〜80% が目安で、step_size を調整して制御します。最初の数百〜数千サンプルは「バーンイン」として捨てます。

ガウス過程（GP）¶

「関数の確率分布」——任意の有限個の点での関数値が多変量ガウス分布に従う確率過程です。

\[ f \sim \mathcal{GP}(m(x), k(x, x')) \]
  • \(m(x)\)：平均関数（事前の予測）
  • \(k(x, x')\)：カーネル関数（点 \(x\) と \(x'\) の関数値の相関）

カーネルの役割¶

カーネル関数が「どんな関数が現れやすいか」を決めます。RBF（ガウス）カーネル \(k(x,x') = \sigma_f^2 \exp\!\left(-\frac{(x-x')^2}{2\ell^2}\right)\) は「近い点の関数値は似ている」という仮定を表します。

GP 回帰の仕組み¶

観測 \(\mathbf{y} = f(X) + \varepsilon\)（\(\varepsilon \sim \mathcal{N}(0, \sigma_n^2 I)\)）に対し、多変量正規分布の条件付き分布の公式から事後分布の予測平均と分散が解析的に計算できます：

\[ \bar{f}_* = K(x_*, X)[K(X,X) + \sigma_n^2 I]^{-1}\mathbf{y} \]
\[ \mathbb{V}[f_*] = K(x_*,x_*) - K(x_*,X)[K(X,X)+\sigma_n^2 I]^{-1}K(X,x_*) \]

観測データが増えるほど \(\mathbb{V}[f_*]\) が小さくなり、不確かさが減ります。scikit-learn の GaussianProcessRegressor で実装できます。

ブラウン運動（ウィーナー過程）¶

ランダムウォークの連続時間極限です。金融モデル・拡散方程式・Diffusion モデルの理論的基盤。

\[ W(t+dt) - W(t) \sim \mathcal{N}(0, dt) \]

性質： - \(W(0) = 0\) - 増分 \(W(t) - W(s)\)（\(t > s\)）は \(\mathcal{N}(0, t-s)\) に従う - 異なる時間区間の増分は独立

np.cumsum(np.random.normal(0, np.sqrt(dt), n_steps)) でシミュレーションできます。Diffusion モデル（画像生成）はブラウン運動を使って画像にノイズを加え、逆過程で復元する枠組みです。

数学的導出¶

ランダムウォークの標準偏差が \(\sqrt{n}\) になる理由¶

\(n\) ステップのランダムウォーク \(X_n = \sum_{k=1}^n Z_k\)（\(Z_k \in \{-1, +1\}\) 等確率）の分散を求めます。

各ステップが独立なので分散は加算できます：

\[ \text{Var}(X_n) = \sum_{k=1}^n \text{Var}(Z_k) = n \cdot \text{Var}(Z_1) \]

\(\text{Var}(Z_1) = E[Z_1^2] - (E[Z_1])^2 = 1 - 0 = 1\) なので：

\[ \text{Var}(X_n) = n \quad \Rightarrow \quad \text{SD}(X_n) = \sqrt{n} \]

これが「拡散係数 \(\sqrt{n}\)」の正体です。金融の「ボラティリティが保有期間の平方根に比例する」という経験則の数学的根拠です。

定常分布の存在条件（詳細つり合い条件）¶

マルコフ連鎖が 詳細つり合い（detailed balance）条件 を満たすとき、定常分布 \(\pi\) が存在します：

\[ \pi_i P_{ij} = \pi_j P_{ji} \quad \forall i, j \]

詳細つり合いが定常分布を保証することの証明：

\[ \sum_i \pi_i P_{ij} = \sum_i \pi_j P_{ji} = \pi_j \sum_i P_{ji} = \pi_j \cdot 1 = \pi_j \]

（各行の和 \(\sum_i P_{ji} = 1\) を使った）

これは \(\pi P = \pi\)、つまり \(\pi\) が定常分布の定義を満たすことを示します。

MCMC との接続：
メトロポリス・ヘイスティングス法の受理確率 \(\min\!\left(1, \frac{\pi(\theta')}{\pi(\theta)}\right)\) は、詳細つり合い条件を満たすように設計されています。これによって、定常分布が目標分布 \(\pi\) になることが保証されます。

ガウス過程の予測分布の導出（スケッチ）¶

観測 \(\mathbf{y} = f(X) + \varepsilon\)（\(\varepsilon \sim \mathcal{N}(0, \sigma_n^2 I)\)）に対し、新しい点 \(x_*\) での予測分布は多変量正規分布の条件付き分布の公式から：

\[ p(f_* \mid X, \mathbf{y}, x_*) = \mathcal{N}(\bar{f}_*, \mathbb{V}[f_*]) \]
\[ \bar{f}_* = K(x_*, X)[K(X,X) + \sigma_n^2 I]^{-1}\mathbf{y} \]
\[ \mathbb{V}[f_*] = K(x_*,x_*) - K(x_*,X)[K(X,X)+\sigma_n^2 I]^{-1}K(X,x_*) \]
  • \(K(x_*, X)\)：新点と観測点の間のカーネル値（類似度）
  • \(\mathbb{V}[f_*]\) は観測が増えると小さくなる → 観測データ周辺で不確かさが減少

確認問題¶

  1. マルコフ性とは何ですか？「天気の明日の予測に 1 週間前の天気は関係ない」はマルコフ性の例ですか？
  2. ランダムウォークの標準偏差が \(\sqrt{n}\) になることを、独立な確率変数の分散の加算性から導いてください。
  3. 詳細つり合い条件 \(\pi_i P_{ij} = \pi_j P_{ji}\) が成り立つとき、\(\pi\) が定常分布であることを証明してください。
