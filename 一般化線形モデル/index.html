一般化線形モデル（GLM）¶

通常の線形回帰を「非正規分布の従属変数」に拡張したモデル群です。ロジスティック回帰（二項分布）・ポアソン回帰（カウントデータ）・ガンマ回帰（正値連続データ）を「尤度 × リンク関数」という統一的な枠組みで記述します。統計検定準1級の頻出領域です。

はじめて読む人へ¶

通常の線形回帰は「結果変数が正規分布に従う」と仮定します。しかし「クリックしたか/しないか（0 or 1）」「1 時間のアクセス数（0 以上の整数）」は正規分布に従いません。GLM はこれらを正しくモデル化する枠組みです。

読む前に押さえること¶

  • 回帰分析 — 線形回帰の基礎・最尤推定
  • 確率・統計基礎 — 確率分布・尤度関数

読み終えたら説明できること¶

  • GLM の 3 要素（確率分布・線形予測子・リンク関数）を説明できる
  • ロジスティック回帰の係数をオッズ比で解釈できる
  • ポアソン回帰とゼロ過剰データの対処法を説明できる

線形回帰の限界¶

問題 1：クリック率（0 or 1 の二値）¶

線形回帰で確率を予測すると 0〜1 の範囲を超えることがあります。

線形回帰で確率を予測すると、\(X\) の値が大きくなるにつれ予測値が 1.0 を超えたり負になったりして、確率としての意味を失います。ロジスティック回帰はシグモイド関数で出力を \((0, 1)\) に収めることでこの問題を解決します。

問題 2：カウントデータ（0 以上の整数）¶

アクセス数・事故件数は負の値にならず、分布が右に歪んでいます。正規分布の仮定は不適切です。

問題 3：等分散の仮定¶

ポアソン分布では「平均 = 分散」——平均が大きいほどばらつきも大きくなります。これは線形回帰の等分散仮定と矛盾します。

GLM の 3 要素¶

\[ \boxed{\underbrace{g(\mu_i)}_{\text{リンク関数}} = \underbrace{\beta_0 + \beta_1 x_{i1} + \cdots + \beta_p x_{ip}}_{\text{線形予測子}\ \eta_i}, \quad \underbrace{Y_i \sim \text{分布族}(\mu_i, \phi)}_{\text{確率分布}}} \]
要素 役割
確率分布 データの生成過程（正規・二項・ポアソン・ガンマなど）
リンク関数 \(g(\cdot)\) 平均 \(\mu\) と線形予測子 \(\eta\) をつなぐ変換
線形予測子 \(\eta\) 通常の線形回帰と同じ \(\mathbf{x}^\top \boldsymbol{\beta}\)

代表的なモデル¶

モデル 確率分布 リンク関数 逆リンク 用途
線形回帰 正規分布 恒等リンク \(g(\mu)=\mu\) \(\mu = \eta\) 連続量
ロジスティック回帰 二項分布 ロジットリンク \(\log\frac{\mu}{1-\mu}\) シグモイド 2 値分類
ポアソン回帰 ポアソン分布 対数リンク \(\log(\mu)\) \(\exp(\eta)\) カウント
ガンマ回帰 ガンマ分布 逆数リンク \(1/\mu\) \(1/\eta\) 正値連続
逆ガウス回帰 逆ガウス分布 \(1/\mu^2\) \((1/\eta)^{1/2}\) 右裾が重い

ロジスティック回帰¶

モデルの形¶

\[ P(Y_i = 1 \mid \mathbf{x}_i) = \sigma(\eta_i) = \frac{1}{1 + e^{-\eta_i}}, \quad \eta_i = \boldsymbol{\beta}^\top \mathbf{x}_i \]
  • \(\sigma(\cdot)\)：シグモイド関数（ロジットリンクの逆）
  • \(\eta\) が大きいほど \(P(Y=1)\) が 1 に近づく

オッズとオッズ比¶

オッズ： \(\frac{p}{1-p}\)（確率 \(p\) に対する「起きる対起きない」の比）

\[ \text{ロジット}: \quad \log\frac{P(Y=1)}{P(Y=0)} = \beta_0 + \beta_1 x_1 + \cdots \]

オッズ比： \(x_j\) が 1 増えたときのオッズの変化倍率：

\[ \text{OR}_j = \exp(\beta_j) \]
\(\hat{\beta}_j\) \(\text{OR}_j\) 解釈
\(< 0\) \(< 1\) \(x_j\) が増えるとイベント確率が下がる
\(= 0\) \(= 1\) \(x_j\) は確率と無関係
\(> 0\) \(> 1\) \(x_j\) が増えるとイベント確率が上がる

例： 「年齢が 1 歳増えるとクリック率が \(\exp(0.3) \approx 1.35\) 倍になる（35% 増）」

尤度と学習¶

\[ L(\boldsymbol{\beta}) = \prod_{i=1}^n \sigma(\eta_i)^{y_i} (1-\sigma(\eta_i))^{1-y_i} \]

負の対数尤度（損失関数）はクロスエントロピー損失と等価です：

\[ -\log L = -\sum_i \left[y_i \log \hat{p}_i + (1-y_i)\log(1-\hat{p}_i)\right] \]

解析解がないため、勾配降下法で最適化します。

多クラス拡張：多項ロジスティック回帰¶

\(K\) クラスへの拡張。Softmax 関数を使います：

\[ P(Y_i = k \mid \mathbf{x}_i) = \frac{\exp(\boldsymbol{\beta}_k^\top \mathbf{x}_i)}{\sum_{j=1}^K \exp(\boldsymbol{\beta}_j^\top \mathbf{x}_i)} \]

ポアソン回帰¶

モデルの形¶

カウントデータ（事故件数・アクセス数・疾患発生数）に対して：

\[ \log(\mu_i) = \boldsymbol{\beta}^\top \mathbf{x}_i \quad \Rightarrow \quad \mu_i = \exp(\boldsymbol{\beta}^\top \mathbf{x}_i) \]

正の値のみ を予測します（対数リンク関数の性質）。

係数の解釈¶

\(x_j\) が 1 増えたとき、期待カウントが \(\exp(\beta_j)\) 倍になります：

\[ \frac{\mu(\mathbf{x} + \mathbf{e}_j)}{\mu(\mathbf{x})} = \exp(\beta_j) \]

例： 「広告費が 1 万円増えるとアクセス数が \(\exp(0.2) \approx 1.22\) 倍（22% 増）になる」

オフセット項¶

観察期間・人口規模が異なる場合に、比率（レート）を同等に比較するためにオフセットを使います：

\[ \log(\mu_i) = \log(E_i) + \boldsymbol{\beta}^\top \mathbf{x}_i \]

\(\log(E_i)\) はオフセット（係数を固定して 1 に固定）。\(\mu_i / E_i\) が実際にモデル化するレートです。

過分散と負の二項分布回帰¶

ポアソン分布の限界¶

ポアソン分布では \(E[Y] = \text{Var}[Y] = \mu\) （平均 = 分散）が仮定されます。

しかし実データでは \(\text{Var}[Y] > \mu\)（過分散）が多い：

実データでは標準偏差が √平均 を大きく上回ること（標準偏差 >> √平均）が多いのに対し、ポアソン分布の理論値は標準偏差 ≈ √平均 を仮定します。 過分散を無視すると、標準誤差が過小推定されて p 値が小さくなりすぎます（偽陽性）。

負の二項分布回帰¶

個体レベルのランダム効果（隠れた異質性）を考慮します：

\[ Y_i \mid \lambda_i \sim \text{Poisson}(\lambda_i), \quad \lambda_i \sim \text{Gamma}(\mu, \theta) \]

\(\lambda_i\) を積分消去すると \(Y_i\) は負の二項分布に従い：

\[ \text{Var}[Y_i] = \mu + \frac{\mu^2}{\theta} \]

\(\theta \to \infty\) でポアソン分布に収束します。

ゼロ過剰モデル（Zero-Inflated）¶

カウントデータで 0 が多すぎる場合（「今日釣れた魚の数」—— 釣りに行かない人は必ず 0）：

\[ P(Y_i = 0) = \pi_i + (1-\pi_i) \cdot p_{\text{Pois}}(0) \]
  • 構造的ゼロ（釣りに行かない）: 確率 \(\pi_i\)
  • サンプリングゼロ（釣りに行ったが釣れない）: ポアソン部分

モデルの評価¶

逸脱度（Deviance）¶

当てはまりの悪さを尤度比で測ります：

\[ D = 2 \left[\log L(\hat{\boldsymbol{\mu}}_{\text{飽和}}) - \log L(\hat{\boldsymbol{\mu}}_{\text{当てはめ}})\right] \]

飽和モデル：各観測値に固有のパラメータを持つ（完全に当てはまる）モデル。

残差逸脱度 \(\sim \chi^2(n - p - 1)\) が大きければモデルの当てはまりが悪い。

AIC（赤池情報量基準）¶

\[ \text{AIC} = -2 \log L + 2p \]

\(p\)：推定したパラメータ数。モデル選択では AIC が小さいモデルを選びます。

擬似 \(R^2\)（McFadden）¶

\[ R^2_{\text{McF}} = 1 - \frac{\log L(\text{当てはめ})}{\log L(\text{切片のみ})} \]

線形回帰の \(R^2\) に相当する当てはまりの指標です（同じく 0〜1 ですが意味は異なります）。

scikit-learn と statsmodels の使い分け¶

ライブラリ 向いている用途 出力
scikit-learn 予測精度重視・ML パイプライン 係数・予測値のみ
statsmodels 統計的推論・係数の解釈 p 値・信頼区間・AIC・逸脱度 
import statsmodels.api as sm

# ポアソン回帰（statsmodels）
X = sm.add_constant(X_data)  # 切片を追加
model = sm.GLM(y, X, family=sm.families.Poisson())
result = model.fit()
print(result.summary())  # 係数・標準誤差・p値・AIC を表示

# ロジスティック回帰
model_logit = sm.GLM(y, X, family=sm.families.Binomial())
result_logit = model_logit.fit()
np.exp(result_logit.params)  # オッズ比

指数型分布族との接続¶

正規・二項・ポアソン・ガンマ・逆ガウスはすべて指数型分布族の特殊ケースです：

\[ f(y; \theta, \phi) = \exp\!\left(\frac{y\theta - b(\theta)}{a(\phi)} + c(y, \phi)\right) \]
  • \(\theta\)：自然パラメータ
  • \(b(\theta)\)：キュムラント関数（\(b'(\theta) = E[Y]\)、\(b''(\theta) = \text{Var}[Y]/a(\phi)\)）

GLM の統一理論はこの構造から生まれています：

\[ E[Y] = \mu = b'(\theta), \quad \text{Var}[Y] = b''(\theta) \cdot a(\phi) \]

数学的導出¶

ロジスティック回帰の勾配¶

負の対数尤度 \(\mathcal{L} = -\sum_i [y_i \log \hat{p}_i + (1-y_i)\log(1-\hat{p}_i)]\) の \(\boldsymbol{\beta}\) に関する勾配：

\[ \frac{\partial \mathcal{L}}{\partial \boldsymbol{\beta}} = -\sum_i (y_i - \hat{p}_i) \mathbf{x}_i = -\mathbf{X}^\top (\mathbf{y} - \hat{\mathbf{p}}) \]

\(\hat{p}_i = \sigma(\mathbf{x}_i^\top \boldsymbol{\beta})\)、\(\sigma' = \sigma(1-\sigma)\) を使います。残差 \((y_i - \hat{p}_i)\) の加重和が勾配になる——線形回帰の最小二乗と同じ構造です。

ヘッシアン \(H = \mathbf{X}^\top W \mathbf{X}\)（\(W = \text{diag}[\hat{p}_i(1-\hat{p}_i)]\)）は半正定値なので、ロジスティック回帰の負の対数尤度は凸関数で唯一の最適解を持ちます。

IRLS（反復重み付き最小二乗法）¶

GLM の推定は MLE と等価で、IRLS アルゴリズムで解きます：

\[ \boldsymbol{\beta}^{(t+1)} = (\mathbf{X}^\top W^{(t)} \mathbf{X})^{-1} \mathbf{X}^\top W^{(t)} \mathbf{z}^{(t)} \]

\(\mathbf{z}^{(t)} = \hat{\boldsymbol{\eta}}^{(t)} + (W^{(t)})^{-1}(\mathbf{y} - \hat{\boldsymbol{\mu}}^{(t)})\) は「作業従属変数」です。各反復で線形回帰を解くことに帰着します。

確認問題¶

  1. ロジスティック回帰の係数 \(\hat{\beta}_j = 0.5\) のとき、オッズ比と「\(x_j\) が 1 増えたときの確率の変化」を説明してください。
  2. ポアソン回帰で過分散が無視されると、何の推定に問題が生じますか？
  3. AIC で「パラメータ数に対するペナルティ」がある理由を説明してください。
  4. GLM のリンク関数が「恒等変換以外が必要な理由」を、二項分布の場合で説明してください。
