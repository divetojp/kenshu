# 対応分析

カテゴリ変数のクロス表（分割表）を低次元空間に可視化する手法です。**行カテゴリと列カテゴリの関係**を同一の散布図上に表示できるのが特徴です。アンケート分析・マーケティング・テキストデータのカテゴリ分析で使います。

---

## はじめて読む人へ

「ブランド A は若年層に、ブランド C は高齢層に好まれている」という関係を、表ではなくグラフで視覚的に掴みたいとき——対応分析は分割表（クロス集計表）を散布図に変換します。近くにプロットされた行と列は「関連が強い」ことを意味します。PCA が数値データを扱うのに対し、対応分析は**カテゴリデータ専用の次元削減**です。

### 読む前に押さえること

- [主成分分析（PCA）](主成分分析.md) — 次元削減の基本的な考え方（対応分析はカテゴリ版 PCA）
- クロス集計表（分割表）の読み方を知っていること

### 読み終えたら説明できること

- 対応分析がどんな形式のデータを入力とするかを説明できる
- 「行と列が近くにプロット」される意味を説明できる
- PCA・MDS・対応分析の使い分けを説明できる

---

## どんなときに使うか

!!! info ""
    分割表の例：ブランド × 購買層
             若年層  中年層  高齢層
    ブランド A   80     30     10
    ブランド B   20     60     40
    ブランド C   10     20     70

「ブランド A は若年層に、ブランド C は高齢層に支持されている」という関係を、散布図上で視覚的に確認できます。

## 実装（prince ライブラリ）

```python
import numpy as np
import pandas as pd
import matplotlib.pyplot as plt

# pip install prince
import prince

# 分割表（ブランド × 購買層）
data = pd.DataFrame(
    [[80, 30, 10],
     [20, 60, 40],
     [10, 20, 70]],
    index=["ブランド A", "ブランド B", "ブランド C"],
    columns=["若年層", "中年層", "高齢層"]
)

ca = prince.CA(n_components=2, random_state=42)
ca = ca.fit(data)

# 行（ブランド）と列（購買層）を同一空間にプロット
ax = ca.plot_coordinates(X=data, figsize=(8, 5))
plt.title("対応分析：ブランド × 購買層")
plt.show()

# 各次元の寄与率
print(ca.eigenvalues_summary)
```

## 内部の数学：特異値分解（SVD）

対応分析の内部では特異値分解（SVD）を使います：

$$
A = U \Sigma V^\top
$$

scipy だけで手動実装する場合：

```python
from scipy.linalg import svd

N = np.array([[80, 30, 10],
              [20, 60, 40],
              [10, 20, 70]], dtype=float)

n = N.sum()
P = N / n                           # 対応行列（確率化）
r = P.sum(axis=1, keepdims=True)    # 行マージン
c = P.sum(axis=0, keepdims=True)    # 列マージン

# 標準化残差行列
S = (P - r @ c) / np.sqrt(r @ c)

# SVD で主座標を取得
U, D, Vt = svd(S, full_matrices=False)

row_coords = np.diag(1 / r.ravel()**0.5) @ U[:, :2] * D[:2]
col_coords = np.diag(1 / c.ravel()**0.5) @ Vt[:2, :].T * D[:2]

# 寄与率
total_inertia = (D**2).sum()
print("次元ごとの寄与率:", (D[:2]**2 / total_inertia).round(3))
```

## MDS との違い

| 観点 | MDS | 対応分析 |
|------|-----|---------|
| 入力 | 距離・類似度行列 | 分割表（度数） |
| 対象 | 量的・質的データ | カテゴリ変数 |
| プロット | 行のみ | 行と列を同時にプロット |
| 用途 | 距離の保存 | カテゴリ間の関連の可視化 |

## 数学的導出

### 対応行列・イナーシャ・カイ二乗統計量の関係

分割表 $N$（$I$ 行 × $J$ 列）から対応分析の基礎となる行列を構築します。

**STEP 1：対応行列**

$$
P_{ij} = \frac{N_{ij}}{n}, \quad n = \sum_{i,j} N_{ij}
$$

行マージン $r_i = \sum_j P_{ij}$、列マージン $c_j = \sum_i P_{ij}$。

**STEP 2：独立の場合の期待値**

行と列が独立なら $P_{ij} = r_i c_j$。実際との差（標準化残差）を：

$$
S_{ij} = \frac{P_{ij} - r_i c_j}{\sqrt{r_i c_j}}
$$

**STEP 3：イナーシャ（total inertia）**

$$
\text{inertia} = \sum_{i,j} \frac{(P_{ij} - r_i c_j)^2}{r_i c_j} = \frac{\chi^2}{n}
$$

イナーシャはカイ二乗統計量 $\chi^2$ を $n$ で割ったものです。**行と列の独立性からの乖離の度合い**を表します。

**STEP 4：SVD で低次元座標を取得**

標準化残差行列 $S$ を SVD 分解：

$$
S = U \Sigma V^\top
$$

行座標：$F = D_r^{-1/2} U \Sigma$（$D_r = \text{diag}(r)$）

列座標：$G = D_c^{-1/2} V \Sigma$（$D_c = \text{diag}(c)$）

**各次元の寄与率** = $\sigma_k^2 / \text{inertia}$（SVD の特異値の二乗の割合）

```python
import numpy as np

N = np.array([[80, 30, 10],
              [20, 60, 40],
              [10, 20, 70]], dtype=float)

n = N.sum()
P = N / n
r = P.sum(axis=1)
c = P.sum(axis=0)

# 標準化残差行列
S = (P - np.outer(r, c)) / np.sqrt(np.outer(r, c))

# SVD
U, D, Vt = np.linalg.svd(S, full_matrices=False)

# イナーシャ = Σ σ² = カイ二乗 / n
inertia = (D**2).sum()
chi2 = inertia * n
print(f"イナーシャ = {inertia:.4f}")
print(f"χ² 統計量 = {chi2:.4f}")
print(f"各次元の寄与率: {(D**2 / inertia).round(3)}")

# 行・列の座標
F = np.diag(1/np.sqrt(r)) @ U[:, :2] * D[:2]
G = np.diag(1/np.sqrt(c)) @ Vt[:2, :].T * D[:2]
print("\n行（ブランド）の座標:", F.round(3))
print("列（購買層）の座標:", G.round(3))
```
