# 対応分析

カテゴリ変数のクロス表（分割表）を低次元空間に可視化する手法です。**行カテゴリと列カテゴリの関係**を同一の散布図上に表示できるのが特徴です。アンケート分析・マーケティング・テキストデータのカテゴリ分析で使います。

---

## どんなときに使うか

```
分割表の例：ブランド × 購買層
         若年層  中年層  高齢層
ブランド A   80     30     10
ブランド B   20     60     40
ブランド C   10     20     70
```

「ブランド A は若年層に、ブランド C は高齢層に支持されている」という関係を、散布図上で視覚的に確認できます。

---

## 実装（prince ライブラリ）

```python
import numpy as np
import pandas as pd
import matplotlib.pyplot as plt

# pip install prince
import prince

# 分割表（ブランド × 購買層）
data = pd.DataFrame(
    [[80, 30, 10],
     [20, 60, 40],
     [10, 20, 70]],
    index=["ブランド A", "ブランド B", "ブランド C"],
    columns=["若年層", "中年層", "高齢層"]
)

ca = prince.CA(n_components=2, random_state=42)
ca = ca.fit(data)

# 行（ブランド）と列（購買層）を同一空間にプロット
ax = ca.plot_coordinates(X=data, figsize=(8, 5))
plt.title("対応分析：ブランド × 購買層")
plt.show()

# 各次元の寄与率
print(ca.eigenvalues_summary)
```

## 内部の数学：特異値分解（SVD）

対応分析の内部では特異値分解（SVD）を使います：

$$
A = U \Sigma V^\top
$$

scipy だけで手動実装する場合：

```python
from scipy.linalg import svd

N = np.array([[80, 30, 10],
              [20, 60, 40],
              [10, 20, 70]], dtype=float)

n = N.sum()
P = N / n                           # 対応行列（確率化）
r = P.sum(axis=1, keepdims=True)    # 行マージン
c = P.sum(axis=0, keepdims=True)    # 列マージン

# 標準化残差行列
S = (P - r @ c) / np.sqrt(r @ c)

# SVD で主座標を取得
U, D, Vt = svd(S, full_matrices=False)

row_coords = np.diag(1 / r.ravel()**0.5) @ U[:, :2] * D[:2]
col_coords = np.diag(1 / c.ravel()**0.5) @ Vt[:2, :].T * D[:2]

# 寄与率
total_inertia = (D**2).sum()
print("次元ごとの寄与率:", (D[:2]**2 / total_inertia).round(3))
```

## MDS との違い

| 観点 | MDS | 対応分析 |
|------|-----|---------|
| 入力 | 距離・類似度行列 | 分割表（度数） |
| 対象 | 量的・質的データ | カテゴリ変数 |
| プロット | 行のみ | 行と列を同時にプロット |
| 用途 | 距離の保存 | カテゴリ間の関連の可視化 |

---

## 手法選択ガイド（多変量解析全体）

| やりたいこと | 手法 | ラベルが必要か |
|------------|------|:------:|
| 次元削減・可視化 | PCA | 不要 |
| クラス分離の最大化 | LDA | 必要 |
| 潜在因子の解釈 | 因子分析 | 不要 |
| グループ分け（k を決める） | k-means | 不要 |
| グループの階層構造を見る | 階層的クラスタリング | 不要 |
| 2 変数群の関係を見る | 正準相関分析 | 不要 |
| 距離行列から地図を作る | MDS | 不要 |
| カテゴリ変数のクロス表を可視化 | **対応分析** | 不要 |
| 非線形な低次元化 | t-SNE / UMAP | 不要 |
