# 正準相関分析（CCA）

2 つの変数群（X の集合と Y の集合）の間の **最大の相関関係** を見つける手法です。「運動能力の指標群」と「生理的指標群」のように、2 種類の測定値セットの関係を分析するときに使います。

---

## はじめて読む人へ

通常の相関係数は「変数 1 本 対 変数 1 本」の関係を見ます。正準相関分析は「変数群 対 変数群」の関係を見ます。「身体能力の指標（走る速さ・握力・跳躍力）」と「健康指標（血圧・体重・心拍数）」がどう関係しているかを一度に分析できます。応用研究・医療・心理学でよく使われる手法です。

### 読む前に押さえること

- [確率・統計基礎](確率・統計基礎.md) の相関・共分散の概念
- [線形代数](線形代数.md) の固有値分解、特に一般化固有値問題
- [回帰分析](回帰分析.md) の重回帰（変数群から 1 変数を予測）との対比で理解が深まります

### 読み終えたら説明できること

- 正準相関分析が「1 変数 対 1 変数」の相関と何が違うかを説明できる
- 正準変量とは何かを、数式を使って説明できる
- CCA が一般化固有値問題に帰着することを説明できる

---

## 問題定義

$X \in \mathbb{R}^{n \times p}$（$p$ 変数群）と $Y \in \mathbb{R}^{n \times q}$（$q$ 変数群）が与えられたとき、線形結合

$$
u = X\mathbf{a}, \quad v = Y\mathbf{b}
$$

の相関係数 $\text{corr}(u, v)$ を最大化する重みベクトル $\mathbf{a} \in \mathbb{R}^p$、$\mathbf{b} \in \mathbb{R}^q$ を求めます。

$$
\max_{\mathbf{a}, \mathbf{b}} \; \text{corr}(Xa, Yb) = \frac{\mathbf{a}^\top \Sigma_{XY} \mathbf{b}}{\sqrt{\mathbf{a}^\top \Sigma_{XX} \mathbf{a}} \sqrt{\mathbf{b}^\top \Sigma_{YY} \mathbf{b}}}
$$

ここで $\Sigma_{XX}$、$\Sigma_{YY}$、$\Sigma_{XY}$ はそれぞれ $X$ の共分散行列、$Y$ の共分散行列、$X$-$Y$ の交差共分散行列です。

---

## 一般化固有値問題への帰着（導出）

### STEP 1：制約条件の標準化

$\mathbf{a}^\top \Sigma_{XX} \mathbf{a} = 1$、$\mathbf{b}^\top \Sigma_{YY} \mathbf{b} = 1$ という制約のもとで $\mathbf{a}^\top \Sigma_{XY} \mathbf{b}$ を最大化します。

ラグランジュ乗数 $\lambda$ を使って偏微分すると：

$$
\frac{\partial}{\partial \mathbf{a}} \left[ \mathbf{a}^\top \Sigma_{XY} \mathbf{b} - \frac{\lambda}{2}(\mathbf{a}^\top \Sigma_{XX} \mathbf{a} - 1) \right] = 0
$$

$$
\Sigma_{XY} \mathbf{b} = \lambda \Sigma_{XX} \mathbf{a} \quad \cdots (1)
$$

同様に $\mathbf{b}$ で偏微分すると：

$$
\Sigma_{YX} \mathbf{a} = \lambda \Sigma_{YY} \mathbf{b} \quad \cdots (2)
$$

### STEP 2：$\mathbf{b}$ を消去して固有値問題に変換

(1) から $\mathbf{b} = \frac{1}{\lambda} \Sigma_{YY}^{-1} \Sigma_{YX} \mathbf{a}$ を (2) に代入すると：

$$
\Sigma_{XX}^{-1} \Sigma_{XY} \Sigma_{YY}^{-1} \Sigma_{YX} \mathbf{a} = \lambda^2 \mathbf{a}
$$

これは行列 $M = \Sigma_{XX}^{-1} \Sigma_{XY} \Sigma_{YY}^{-1} \Sigma_{YX}$ の **固有値問題** です。

$$
\boxed{M\mathbf{a} = \rho^2 \mathbf{a}}, \quad \rho^2 = \lambda^2 \text{ が正準相関係数の二乗}
$$

固有値の平方根 $\rho_1 \geq \rho_2 \geq \cdots$ が **正準相関係数** で、対応する固有ベクトルが **正準重みベクトル** $\mathbf{a}_1, \mathbf{a}_2, \ldots$ です。

---

## 正準変量とは

第 1 正準変量のペア $(u_1, v_1)$ は：

$$
u_1 = X\mathbf{a}_1 \quad (\text{X 群の線形結合}), \qquad v_1 = Y\mathbf{b}_1 \quad (\text{Y 群の線形結合})
$$

- $\text{corr}(u_1, v_1) = \rho_1$（最大の正準相関係数）
- 第 2 正準変量 $(u_2, v_2)$ は $(u_1, v_1)$ と無相関という制約のもとで次に高い相関を持つ

---

## 実装

```python
import numpy as np
from sklearn.cross_decomposition import CCA
from sklearn.datasets import load_linnerud

# Linnerud データ：運動能力(X) × 生理指標(Y)
data = load_linnerud()
X, Y = data.data, data.target

cca = CCA(n_components=2)
X_c, Y_c = cca.fit_transform(X, Y)

# 第 1・第 2 正準変量の相関
r1 = np.corrcoef(X_c[:, 0], Y_c[:, 0])[0, 1]
r2 = np.corrcoef(X_c[:, 1], Y_c[:, 1])[0, 1]
print(f"第 1 正準相関係数: {r1:.3f}")
print(f"第 2 正準相関係数: {r2:.3f}")
```

### 手計算で確認

```python
import numpy as np
from sklearn.preprocessing import StandardScaler
from sklearn.datasets import load_linnerud

data = load_linnerud()
X = StandardScaler().fit_transform(data.data)
Y = StandardScaler().fit_transform(data.target)

n = X.shape[0]
Sigma_XX = X.T @ X / n
Sigma_YY = Y.T @ Y / n
Sigma_XY = X.T @ Y / n
Sigma_YX = Sigma_XY.T

# M = Sigma_XX^{-1} Sigma_XY Sigma_YY^{-1} Sigma_YX
inv_XX = np.linalg.inv(Sigma_XX)
inv_YY = np.linalg.inv(Sigma_YY)
M = inv_XX @ Sigma_XY @ inv_YY @ Sigma_YX

eigenvalues = np.linalg.eigvals(M).real
rho_squared = np.sort(eigenvalues)[::-1]
print("正準相関係数の二乗:", rho_squared.round(4))
print("正準相関係数:", np.sqrt(np.maximum(rho_squared, 0)).round(4))
```

## 他の手法との使い分け

| やりたいこと | 手法 |
|------------|------|
| 1 変数 対 1 変数の相関 | ピアソン相関・スピアマン相関 |
| 1 変数群 → 1 変数の予測 | 重回帰分析 |
| **変数群 対 変数群の関係** | **正準相関分析（CCA）** |
| 変数群の潜在構造を見たい | 因子分析 |
| 変数群の教師あり次元削減 | 部分最小二乗法（PLS） |
