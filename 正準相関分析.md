# 正準相関分析（CCA）

2 つの変数群（X の集合と Y の集合）の間の **最大の相関関係** を見つける手法です。「運動能力の指標群」と「生理的指標群」のように、2 種類の測定値セットの関係を分析するときに使います。

---

## 仕組み

通常の相関分析は変数 1 対 1 の関係を見ますが、CCA は **変数群 対 変数群** の関係を見ます。

$$
\text{maximize} \quad \text{corr}(Xa, Yb) \quad \text{subject to} \quad \|a\| = \|b\| = 1
$$

$a$, $b$ は各変数群の重みベクトルで、この線形結合（正準変量）同士の相関が最大になるよう求めます。

---

## 実装

```python
import numpy as np
from sklearn.cross_decomposition import CCA
from sklearn.datasets import load_linnerud

# Linnerud データ：運動能力(X) × 生理指標(Y)
data = load_linnerud()
X, Y = data.data, data.target

cca = CCA(n_components=2)
X_c, Y_c = cca.fit_transform(X, Y)

# 第 1・第 2 正準変量の相関
r1 = np.corrcoef(X_c[:, 0], Y_c[:, 0])[0, 1]
r2 = np.corrcoef(X_c[:, 1], Y_c[:, 1])[0, 1]
print(f"第 1 正準相関係数: {r1:.3f}")
print(f"第 2 正準相関係数: {r2:.3f}")
```

---

## 他の手法との使い分け

| やりたいこと | 手法 |
|------------|------|
| 1 変数 対 1 変数の相関 | ピアソン相関・スピアマン相関 |
| 1 変数群 → 1 変数の予測 | 重回帰分析 |
| **変数群 対 変数群の関係** | **正準相関分析（CCA）** |
| 変数群の潜在構造を見たい | 因子分析 |
