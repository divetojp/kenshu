# 確率過程

時間とともに変化する**確率的な現象**を数学的に扱うフレームワークです。「株価の変動」「ランダムウォーク」「ニューロンの発火」——これらは時刻ごとに確率変数が定まる確率過程です。機械学習では **MCMC**（ベイズ推論のサンプリング）・**ガウス過程回帰**・強化学習の **MDP** の基盤になります。

---

## はじめて読む人へ

確率変数が「1 つの値」ではなく「時系列」として続く場面が確率過程です。ベイズ理論で学んだ MCMC は「マルコフ連鎖」を使ったサンプリング手法、ガウス過程回帰は「無限次元のガウス分布」——どちらもこのページの概念が土台です。

### 読む前に押さえること

- [確率・統計基礎](確率・統計基礎.md) の条件付き確率・確率変数・期待値
- [ベイズ理論](ベイズ理論.md) の事前・事後分布（MCMC との接続のため）

### 読み終えたら説明できること

- マルコフ連鎖の「現在の状態だけで未来が決まる」性質を説明できる
- MCMC が事後分布からのサンプリングに使われる仕組みを説明できる
- ガウス過程が「関数の確率分布」であることを説明できる

## 確率過程とは

時刻 $t$ において確率変数 $X_t$ が定まる族 $\{X_t\}_{t \geq 0}$（または $\{X_n\}_{n=0,1,2,\ldots}$）です。

```
1 つの確率変数:  X ~ N(0, 1)  →  1 つの数値をランダムに引く

確率過程:
  t=0: X₀ = 0
  t=1: X₁ = 0.3  （X₀ から確率的に変化）
  t=2: X₂ = -0.1
  t=3: X₃ = 0.7
  ...
  各時刻で確率変数がある、その無限次元の列
```

## ランダムウォーク

最もシンプルな確率過程です。各ステップで +1 か -1 に等確率で動きます。

$$
X_n = X_0 + \sum_{k=1}^{n} Z_k, \quad Z_k \sim \text{Uniform}(\{-1, +1\})
$$

```python
import numpy as np
import matplotlib.pyplot as plt

np.random.seed(42)
n_steps = 1000
n_paths = 5

plt.figure(figsize=(10, 5))
for _ in range(n_paths):
    steps = np.random.choice([-1, 1], size=n_steps)
    path = np.cumsum(steps)
    plt.plot(path, alpha=0.7)

plt.axhline(0, color='black', linewidth=0.5)
plt.title(f"ランダムウォーク（{n_paths} 本）")
plt.xlabel("ステップ数")
plt.ylabel("位置")
plt.savefig("random_walk.png", dpi=150)

# 標準偏差は √n で増える（拡散）
path = np.cumsum(np.random.choice([-1, 1], size=10000))
print(f"1000 ステップ後の標準偏差 ≈ {np.std([np.sum(np.random.choice([-1,1], size=1000)) for _ in range(1000)]):.1f}")
print(f"理論値 √1000 ≈ {1000**0.5:.1f}")
```

## マルコフ連鎖

**現在の状態だけが次の状態を決める**（過去は不要）性質を**マルコフ性**といいます。

$$
P(X_{n+1} = j \mid X_n = i, X_{n-1}, \ldots, X_0) = P(X_{n+1} = j \mid X_n = i) = P_{ij}
$$

### 遷移行列

```python
import numpy as np

# 天気の Markov 連鎖: 晴=0, 曇=1, 雨=2
# P[i][j] = 状態 i から状態 j への遷移確率
P = np.array([
    [0.7, 0.2, 0.1],   # 晴 → 晴:70%, 曇:20%, 雨:10%
    [0.3, 0.4, 0.3],   # 曇 → 晴:30%, 曇:40%, 雨:30%
    [0.2, 0.3, 0.5],   # 雨 → 晴:20%, 曇:30%, 雨:50%
])

# n ステップ後の遷移行列
def n_step_transition(P, n):
    result = np.eye(len(P))
    for _ in range(n):
        result = result @ P
    return result

print("10 ステップ後の遷移行列:")
print(n_step_transition(P, 10).round(3))

# 定常分布（長期的な各状態の確率）
# π P = π を解く
eigenvalues, eigenvectors = np.linalg.eig(P.T)
stationary = eigenvectors[:, np.argmax(np.isclose(eigenvalues, 1))]
stationary = np.real(stationary / stationary.sum())
print(f"\n定常分布（長期的な天気の割合）: {stationary.round(3)}")
```

### シミュレーション

```python
def simulate_markov_chain(P, initial_state, n_steps):
    states = [initial_state]
    for _ in range(n_steps):
        current = states[-1]
        next_state = np.random.choice(len(P), p=P[current])
        states.append(next_state)
    return states

states = simulate_markov_chain(P, initial_state=0, n_steps=1000)
state_counts = np.bincount(states) / len(states)
print(f"シミュレーション比率: {state_counts.round(3)}")
# → 定常分布に収束する
```

## MCMC（マルコフ連鎖モンテカルロ）

直接サンプリングが難しい複雑な確率分布から、**マルコフ連鎖を使ってサンプリング**する手法です。ベイズ推論で事後分布 $p(\theta | \text{data})$ からのサンプルを得るために使われます。

### メトロポリス・ヘイスティングス法

```python
import numpy as np
import matplotlib.pyplot as plt

def metropolis_hastings(target_log_prob, init, n_samples, step_size=0.5):
    """
    target_log_prob: サンプルしたい分布の対数確率密度
    init: 初期値
    """
    samples = [init]
    current = init
    accepted = 0

    for _ in range(n_samples):
        # 提案分布（正規分布）から新しい候補を生成
        proposal = current + np.random.normal(0, step_size)

        # 受理確率: min(1, p(proposal)/p(current))
        log_accept = target_log_prob(proposal) - target_log_prob(current)

        if np.log(np.random.uniform()) < log_accept:
            current = proposal   # 受理
            accepted += 1
        samples.append(current)  # 棄却でも現在値を追加

    print(f"受理率: {accepted / n_samples:.2%}")
    return np.array(samples)

# 目標分布: N(3, 1) と N(-1, 1.5) の混合
def log_target(x):
    return np.log(0.4 * np.exp(-0.5*(x-3)**2) + 0.6 * np.exp(-0.5*((x+1)/1.5)**2))

samples = metropolis_hastings(log_target, init=0.0, n_samples=10000)

plt.figure(figsize=(10, 4))
plt.subplot(1, 2, 1)
plt.plot(samples[:500], alpha=0.7)
plt.title("MCMC トレース（最初の 500 ステップ）")
plt.xlabel("ステップ")

plt.subplot(1, 2, 2)
plt.hist(samples[1000:], bins=50, density=True, alpha=0.7, label="MCMC サンプル")
x_range = np.linspace(-6, 8, 300)
true_density = np.exp([log_target(x) for x in x_range])
true_density /= np.trapz(true_density, x_range)
plt.plot(x_range, true_density, 'r-', label="真の分布")
plt.legend()
plt.title("サンプル分布 vs 真の分布")
plt.tight_layout()
plt.savefig("mcmc.png", dpi=150)
```

## ガウス過程（GP）

「関数の確率分布」——任意の有限個の点での関数値が**多変量ガウス分布**に従う確率過程です。

$$
f \sim \mathcal{GP}(m(x), k(x, x'))
$$

- $m(x)$：平均関数（事前の予測）
- $k(x, x')$：カーネル関数（点 $x$ と $x'$ の関数値の相関）

```python
import numpy as np
import matplotlib.pyplot as plt
from scipy.stats import multivariate_normal

# RBF（ガウス）カーネル
def rbf_kernel(x1, x2, length_scale=1.0, sigma_f=1.0):
    dist_sq = (x1[:, None] - x2[None, :])**2
    return sigma_f**2 * np.exp(-dist_sq / (2 * length_scale**2))

# テスト点
x_test = np.linspace(-3, 3, 100)

# 事前分布からのサンプル（観測なし）
K = rbf_kernel(x_test, x_test) + 1e-6 * np.eye(len(x_test))
samples = np.random.multivariate_normal(np.zeros(len(x_test)), K, size=5)

plt.figure(figsize=(10, 4))
plt.subplot(1, 2, 1)
for s in samples:
    plt.plot(x_test, s, alpha=0.6)
plt.title("GP 事前分布からのサンプル")
plt.xlabel("x"); plt.ylabel("f(x)")
```

### GP 回帰（観測を条件付け）

```python
# 観測データ
x_train = np.array([-2, -1, 0, 1, 2], dtype=float)
y_train = np.sin(x_train) + 0.1 * np.random.randn(len(x_train))
noise = 0.1

# カーネル行列
K_tt = rbf_kernel(x_train, x_train) + noise**2 * np.eye(len(x_train))
K_ss = rbf_kernel(x_test, x_test)
K_st = rbf_kernel(x_test, x_train)

# 事後分布の平均と分散（解析解）
K_tt_inv = np.linalg.inv(K_tt)
mu_post  = K_st @ K_tt_inv @ y_train
var_post = np.diag(K_ss - K_st @ K_tt_inv @ K_st.T)

plt.subplot(1, 2, 2)
plt.plot(x_test, np.sin(x_test), 'k--', label="真の関数 sin(x)")
plt.plot(x_test, mu_post, 'b-', label="GP 予測平均")
plt.fill_between(x_test,
                  mu_post - 2*np.sqrt(var_post),
                  mu_post + 2*np.sqrt(var_post),
                  alpha=0.3, label="±2σ 信頼区間")
plt.scatter(x_train, y_train, c='red', zorder=5, label="観測データ")
plt.legend(); plt.title("GP 回帰")
plt.tight_layout()
plt.savefig("gp.png", dpi=150)
```

## ブラウン運動（ウィーナー過程）

ランダムウォークの連続時間極限です。金融モデル・拡散方程式・Diffusion モデルの理論的基盤。

$$
W(t+dt) - W(t) \sim \mathcal{N}(0, dt)
$$

```python
def brownian_motion(T=1.0, n_steps=1000, n_paths=5):
    dt = T / n_steps
    t = np.linspace(0, T, n_steps + 1)
    paths = np.zeros((n_paths, n_steps + 1))
    for i in range(n_paths):
        increments = np.random.normal(0, np.sqrt(dt), n_steps)
        paths[i] = np.concatenate([[0], np.cumsum(increments)])
    return t, paths

t, paths = brownian_motion()
plt.figure(figsize=(8, 4))
for path in paths:
    plt.plot(t, path, alpha=0.7)
plt.title("ブラウン運動（5 本の経路）")
plt.xlabel("時間 t"); plt.ylabel("W(t)")
plt.savefig("brownian.png", dpi=150)
```
