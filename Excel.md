## データ分析で使う関数

### 集計関数

| 関数 | 使い方 | 例 |
|------|--------|-----|
| `SUM` | 合計 | `=SUM(B2:B100)` |
| `AVERAGE` | 平均 | `=AVERAGE(B2:B100)` |
| `COUNT` | 数値の個数 | `=COUNT(B2:B100)` |
| `COUNTA` | 空白以外の個数 | `=COUNTA(A2:A100)` |
| `MAX / MIN` | 最大・最小 | `=MAX(B2:B100)` |
| `MEDIAN` | 中央値 | `=MEDIAN(B2:B100)` |

### 条件付き集計（重要）

!!! info ""
    ```
    =COUNTIF(範囲, 条件)
    =COUNTIF(C2:C100, "東京")          → 「東京」の件数
    
    =SUMIF(条件範囲, 条件, 合計範囲)
    =SUMIF(C2:C100, "東京", D2:D100)   → 「東京」の売上合計
    
    =AVERAGEIF(条件範囲, 条件, 平均範囲)
    =AVERAGEIF(B2:B100, "男", C2:C100) → 「男」の平均年収
    
    =COUNTIFS(条件範囲1, 条件1, 条件範囲2, 条件2, ...)
    =COUNTIFS(C2:C100,"東京", D2:D100,">1000000")
    → 「東京」かつ「売上 > 100万」の件数
    ```

### 条件分岐

!!! info ""
    ```
    =IF(条件, 真の場合, 偽の場合)
    =IF(B2>=60, "合格", "不合格")
    
    =IFS(条件1, 値1, 条件2, 値2, ...)   ← 複数条件（Excel 2019以降）
    =IFS(B2>=90,"優", B2>=70,"良", B2>=60,"可", TRUE,"不可")
    ```

### 検索・参照（VLOOKUP / XLOOKUP）

!!! info ""
    ```
    =VLOOKUP(検索値, 範囲, 列番号, 完全一致)
    =VLOOKUP(E2, A2:C100, 2, FALSE)
    → E2 の値を A 列で探し、見つかった行の B 列（2列目）の値を返す
    
    =XLOOKUP(検索値, 検索範囲, 返却範囲)  ← 新しい関数（推奨）
    =XLOOKUP(E2, A2:A100, B2:B100)
    ```

**pandas との対応：**
```python
# VLOOKUP の Python 版
result = df_left.merge(df_right, on='id', how='left')
```

## ピボットテーブル（クロス集計）

複数の軸でデータを集計する最も強力な機能です。

### 作成手順

1. データ範囲をクリック → **挿入** → **ピボットテーブル**
2. 行・列・値・フィルターにフィールドをドラッグ

**例：販売データを「地域 × 商品カテゴリ」別に売上合計を集計**

!!! info ""
    ```
    【元データ】
    日付, 地域, 商品カテゴリ, 売上
    2024/01/01, 東京, 食品, 15000
    2024/01/01, 大阪, 電子機器, 80000
    ...
    
    【ピボットテーブル】
            食品    電子機器   衣料
    東京    45,000  230,000   67,000
    大阪    38,000  195,000   52,000
    名古屋  29,000  145,000   41,000
    ```

**pandas との対応：**
```python
# ピボットテーブルの Python 版
pivot = df.pivot_table(
    values='売上',
    index='地域',
    columns='商品カテゴリ',
    aggfunc='sum'
)
```

## グラフ作成

### グラフの選び方

| データの性質 | 適切なグラフ |
|------------|------------|
| 量の比較（カテゴリ間） | 棒グラフ |
| 推移（時系列） | 折れ線グラフ |
| 割合・構成比 | 円グラフ・積み上げ棒グラフ |
| 2 変数の関係 | 散布図 |
| 分布 | ヒストグラム |

### 作成手順

1. グラフにするデータ範囲を選択
2. **挿入** → グラフの種類を選択
3. グラフを選択した状態で**デザイン**タブから調整

**良いグラフの基本：**
- 必ずタイトルをつける
- 軸ラベルと単位を明記する
- 凡例を適切に配置する
- 3D グラフは視覚的に歪むため避ける

## データのクレンジング

分析の前に「汚いデータ」を整えます。

### よくある問題と対処

!!! info ""
    ```
    【空白の除去】
    =TRIM(A2)   → 前後・中間の余分なスペースを削除
    
    【文字列の統一】
    =LOWER(A2)  → 小文字に統一
    =UPPER(A2)  → 大文字に統一
    
    【文字列の分割】
    データ → 区切り位置 → カンマ・スペースで分割
    
    【重複の削除】
    データ → 重複の削除
    
    【異常値の発見】
    条件付き書式 → 上位/下位 10% を色付け
    ```

!!! info ""
    ```
    【日付の書式統一】
    2024年1月1日 → 2024/1/1 に統一
    TEXT(A2, "YYYY/MM/DD")
    
    【数値として入力された文字列】
    '1000 のように ' がついている → VALUE(A2) で数値に変換
    ```

## Excel から Python への橋渡し

Excel でやっている操作がそのまま pandas の操作に対応します。

| Excel の操作 | pandas の操作 |
|------------|--------------|
| フィルター | `df[df['列'] == '条件']` |
| ソート | `df.sort_values('列')` |
| SUMIF | `df.groupby('グループ列')['値列'].sum()` |
| VLOOKUP | `df1.merge(df2, on='キー')` |
| ピボットテーブル | `df.pivot_table(...)` |
| 条件付き書式 | matplotlib / seaborn でヒートマップ |

```python
import pandas as pd

# Excel ファイルの読み込み
df = pd.read_excel('data.xlsx', sheet_name='Sheet1')

# Excel への書き出し
df.to_excel('output.xlsx', index=False)

# 複数シートへの書き出し
with pd.ExcelWriter('output.xlsx') as writer:
    df_summary.to_excel(writer, sheet_name='集計')
    df_detail.to_excel(writer, sheet_name='詳細')
```
